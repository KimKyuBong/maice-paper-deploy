## 🎯 Pearson r=0.709의 정확한 의미

### ❓ "누구와의 상관관계?"

**답**: 3개 LLM 모델 간의 상호 일치도

### 계산 방법

```
3개 모델 쌍별 상관:
1. OpenAI ↔ Anthropic: r₁
2. OpenAI ↔ Gemini: r₂  
3. Anthropic ↔ Gemini: r₃

평균 상관계수:
Pearson r = (r₁ + r₂ + r₃) / 3 = 0.709
```

### 해석

- **r=0.709*** (p<0.001)
- 3개 독립 AI 모델이 서로 70.9% 수준으로 일치
- "높은 모델 간 일치도" 달성
- 평가의 **객관성과 일관성** 확보

### ⚠️ 주의: 교사와의 상관 아님!

- **교사와의 상관**: r=0.627~0.670 (표Ⅴ-13 참조)
- **모델 간 상관**: r=0.709 (신뢰도 지표)
- 서로 다른 개념!
