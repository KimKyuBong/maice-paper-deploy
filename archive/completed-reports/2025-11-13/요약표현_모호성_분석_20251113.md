# 요약표현 모호성 분석 (2025.11.13)

## 🔍 문제점 발견

### 대화3 (세션 73) - "[2-3] 학생-MAICE" 표현

**논문 표현**:
```
[2-3] | 학생-MAICE | (구체적 문제: $1+2+\cdots+n=\frac{n(n+1)}{2}$) | - | 예시 적용
```

**실제 원문**:
- [2] **MAICE**: "n=k+1 단계가 어려운 게, 식을 어떻게 전개해야 할지 몰라서인가요? 아니면 가정 부분을 어떻게 써야 할지 헷갈리신 건가요? 🤔"
- [3] **학생**: "둘 다"

**모호한 이유**:
1. ❌ **순서 반대**: "학생-MAICE"라고 했지만 실제로는 [2]가 **MAICE**, [3]이 **학생**
2. ❌ **내용 불일치**: 논문은 "구체적 문제 제시"라고 했지만, 실제로는 [2]가 **명료화 질문**, [3]이 **"둘 다"**라는 수동적 응답
3. ❌ **핵심 누락**: 실제 [2]는 **명료화 질문**인데, 논문에는 "구체적 문제"로 잘못 표현됨

**실제 구체적 문제 제시는**:
- [5] 학생: "1*2+2*3+3*4+...+n(n+1) = n(n+1)(n+2)/3"
- [6] MAICE: 해당 문제 증명 설명

---

### 대화3 (세션 73) - "[6-7] 학생-MAICE" 표현

**논문 표현**:
```
[6-7] | 학생-MAICE | "다른 문제도 해봐?" → 등비급수 문제 제시 | - | 전이 시도
```

**실제 원문**:
- [6] **MAICE**: "$1\cdot2 + 2\cdot3 + 3\cdot4 + \cdots + n(n+1) = \frac{n(n+1)(n+2)}{3}$" 증명 설명
- [7] **학생**: "그럼 1+2+2^2+...+2^(n-1)=2^n-1 은?"

**모호한 이유**:
1. ❌ **순서 반대**: "학생-MAICE"라고 했지만 실제로는 [6]이 **MAICE**, [7]이 **학생**
2. ❌ **내용 불일치**: 논문은 "다른 문제도 해봐?"라고 했지만, 실제로는 [7] 학생이 **등비급수 문제를 직접 제시**
3. ⚠️ **"전이 시도" 해석**: 학생이 스스로 다른 문제를 제시한 것은 맞지만, "다른 문제도 해봐?"라는 질문은 없음

### 대화5 (세션 125) - "[2-3] 학생-MAICE" 표현

**논문 표현**:
```
[2-3] | 학생-MAICE | "좀 더 상세하게" | - | 깊이 요구
```

**실제 원문**:
- [2] **MAICE**: 다른 문제 제시 (n! > 2^n 증명)
- [3] **학생**: "4 이상의 모든 자연수 n에 대해, n! > 2^n임을 수학적 귀납법을 이용하여 증명하시오" (같은 문제 반복)
- [4] **학생**: "귀납 단계 더 자세히 설명좀" ← **이게 "좀 더 상세하게" 요구**

**모호한 이유**:
1. ❌ **범위 오류**: "[2-3]"이라고 했지만, "좀 더 상세하게" 요구는 실제로 **[4]**에 있음
2. ❌ **내용 불일치**: [2-3]은 실제로는 **다른 문제 제시 및 반복**인데, 논문은 "좀 더 상세하게" 요구로 표현
3. ❌ **핵심 누락**: [2] MAICE가 다른 문제를 제시한 것이 논문에 반영되지 않음

---


**실제 원문**:
- [4] **학생**: "귀납 단계 더 자세히 설명좀"
- [5] **MAICE**: 귀납 단계 상세 설명

**모호한 이유**:
1. ⚠️ **범위 문제**: "[4-5] MAICE"라고 했지만, [4]는 **학생** 발화
2. ✅ **내용 일치**: [5] MAICE의 "귀납 단계 상세 설명"은 맞음
3. ⚠️ **학생 요구 누락**: [4] 학생의 "더 자세히 설명좀" 요구가 표에 명시되지 않음

## 📊 종합 분석

### 모호성의 원인

1. **순서 표기 오류**: "학생-MAICE"는 학생이 먼저 말한 것처럼 보이지만, 실제로는 MAICE가 먼저 말함
2. **범위 표기 오류**: "[2-3]"이라고 했지만 실제 내용은 [2-4]에 걸쳐 있음
3. **내용 요약 오류**: 실제 대화 내용과 논문 요약이 다름
4. **핵심 정보 누락**: 명료화 질문, 다른 문제 제시 등 중요한 맥락이 누락됨

| 논문 표현 | 실제 대화 | 문제점 |
|----------|----------|--------|
| "[2-3] 학생-MAICE" | [2] MAICE 명료화, [3] 학생 "둘 다" | 순서 반대, 내용 불일치 |
| "[6-7] 학생-MAICE" | [6] MAICE 설명, [7] 학생 문제 제시 | 순서 반대, 질문 없음 |
| "[2-3] 학생-MAICE '좀 더 상세하게'" | [2] MAICE 문제 제시, [3] 학생 반복, [4] 학생 "더 자세히" | 범위 오류, 내용 불일치 |
| "[4-5] MAICE" | [4] 학생 요구, [5] MAICE 설명 | 범위 오류 (학생 포함) |
