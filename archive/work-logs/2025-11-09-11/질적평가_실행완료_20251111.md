## 💡 논문 적용 인사이트

### 인사이트 1: 명료화가 Bloom 고차원 유도

**기존 양적 결과**:
- Agent 모드: +2.25점 (p=0.031)

**질적 증거**:
- 우수 세션 4개 모두 `agent` 모드
- 명료화 → 질문 명확화 → AI 답변 깊이 ↑ → Bloom 단계 ↑

### 인사이트 2: 하위권(T1)의 특수성

**세션 414 (T1, 31.33점)**:
- 성적 하위권이지만 **가장 높은 점수 중 하나**
- 9턴의 깊이 있는 대화
- **Evaluate 단계 도달** (고차원 사고)

**해석**:
- 성적과 관계없이, 적극적 참여 + 명료화 = 높은 학습 효과
- 이것이 바로 "Q1 하위권 +6.91점 효과"의 실제 증거

### 인사이트 3: 점수 구간별 교육적 특성

| 구간 | Bloom 단계 | 대화 특성 | 교육적 의미 |
|------|-----------|----------|------------|
| 우수(30-34) | Evaluate, Analyze | 깊이, 반복, 검증 | 고차원 사고 촉진 |
| 중간(20-29) | Apply | 절차 학습 | 문제 해결 기술 습득 |
| 하위(10-19) | 답변 없음 | 명료화만 | 시스템 사용법 미숙 |
| 최하위(<10) | 없음 | 학습 의도 없음 | 제외 필요 |

#### [사례 1] 하위권 학생의 고차원 사고 도달 (세션 414)

**기본 정보**:
- 점수: 31.33점 (상위 15%)
- 성적 분위: T1 (하위 33%)
- 대화: 9턴, K3 → 고차원 사고

**대화 흐름**:
1. 학생: "수학적 귀납법 $n < n^2$ 증명에서 $k+1$ 단계 질문"
2. AI: K3 분류, 절차적 안내
3. 학생: "서술형 평가 형식으로 다시 설명해줘"
4. AI: 더 구조화된 설명 (Evaluate 단계)
5. 학생: 반복 질문으로 완전 이해 확인

**Bloom 단계**: Understand → Apply → Analyze → **Evaluate**

**교육적 의의**:
- 성적 하위권이지만 AI와의 깊이 있는 대화로 고차원 사고 도달
- 이것이 "Q1 하위권 +6.91점" 효과의 실제 메커니즘
- 명료화와 반복 질문이 핵심

#### [사례 2] 체계적 개념 학습 (세션 150)

**학습 주제**: "2의 거듭제곱 합"  
**핵심 개념**: 등비급수 합 공식, 수학적 귀납법

**대화 흐름**:
1. K2 (개념): "수학적 귀납법의 기저단계와 귀납단계는?"
2. AI: 개념 설명 (Remember, Understand)
3. K3 (적용): "등비급수 합 공식 증명"
4. AI: 오류 지적 + 올바른 증명 (Apply)

**Bloom 단계**: Remember → Understand → **Apply**

**교육적 의의**:
- K2 → K3 자연스러운 진행
- AI가 학생 오류 발견 및 교정
- 체계적 학습 지원
