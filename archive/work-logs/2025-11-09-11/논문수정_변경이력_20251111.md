```markdown
3. 가설 형성 (방법 탐색)
```

```markdown
#### (1) 점수와 Bloom 단계의 강한 연관

분석 결과, LLM 평가점수는 교육적으로 의미 있는 지표임이 확인되었다.

**증거 1: 고득점 세션의 Bloom 고차원 도달**
- 30점 이상 (N=4): 100% Analyze 이상, 75% Evaluate 도달
- 20-29점 (N=3): 100% Apply 이상
- 20점 미만 (N=2): 0% 교육적 답변

**증거 2: 대화 깊이와 점수의 관계**
- 30점 이상: 평균 5.25턴
- 20-29점: 평균 3.0턴
- 20점 미만: 평균 1.5턴

**증거 3: Dewey 반성적 사고 구현도**
- 30점 이상: 평균 4.0/5 단계 (80%)
- 20-29점: 평균 2.3/5 단계 (46%)
- 20점 미만: 평균 0.7/5 단계 (14%)

#### (2) "AI가 AI 평가" 순환 논리의 해소

**기존 우려**: 
LLM이 LLM 피드백을 평가하는 것이 순환 논리가 아닌가?

**본 연구의 답변**:
LLM 평가점수는 다음 3가지 독립적 증거로 타당성이 검증되었다:

1. **교사 평가와의 높은 일치** (2-라절):
   - 상관계수 r=0.743 (p<0.001)
   - 하위권 효과: LLM +2.46점, 교사 +6.91점 (방향 일치)

2. **Bloom/Dewey 이론과의 정합성** (4-나, 다절):
   - 고득점 세션 = Bloom 고차원 + Dewey 완성도 높음
   - 저득점 세션 = 교육적 상호작용 부재

3. **학생 자기 평가와의 수렴** (3절):
   - 68.6%가 질문 유도형(Agent) 선호
   - 이유: "사고력 향상", "깊은 이해"

**결론**: 
LLM 평가점수는 단순 자동 채점이 아닌, 
**Bloom 교육 목표 달성도**와 **Dewey 반성적 사고 구현도**를 
반영하는 교육적으로 타당한 지표이다.

#### (3) 하위권(Q1) 효과의 질적 메커니즘

**양적 발견** (2-나-(3)):
- Q1 하위권 Agent 모드: +2.46점 (LLM), +6.91점 (교사)
- Cohen's d = 0.511~1.117 (중간~큰 효과)

**질적 메커니즘** (세션 414 기반):

```
하위권 학생 (T1, 중간고사 44.4점)
  ↓
불완전한 초기 질문 ("$k+1$일 때 어떻게?")
  ↓
명료화 3회 (메타인지 촉진)
  ↓
9턴의 깊이 있는 대화
  ↓
Bloom 4단계 상승 (Understand → Evaluate)
  ↓
Dewey 5단계 완전 구현
  ↓
31.33점 달성 (+5.06점, 상위 15%)
```

**메커니즘 요약**:
1. 명료화 → 질문 구체화 (Dewey 문제 정의)
2. 반복 대화 → 점진적 비계 (Vygotsky ZPD)
3. 다층 설명 → Bloom 단계 상승
4. 메타 질문 → 고차원 사고 (Evaluate)

이는 단순 통계적 차이가 아닌, **교육학적으로 설명 가능한 메커니즘**이다.
```

**변경 이유**: 
- 양적 결과의 "왜"를 질적으로 규명
- 통계와 사례의 직접 연결
