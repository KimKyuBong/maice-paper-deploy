## 📝 수정된 주요 부분 (3곳)

### 1. 국문 초록 (새로 추가)

```markdown
본 연구의 궁극적 목표는 단순히 효과적인 AI 도구를 개발하는 것을 넘어, 
교사가 AI 교육 도구의 수동적 소비자가 아닌 능동적 설계자이자 연구자로서 
자신의 교육 맥락에 최적화된 AI 시스템을 직접 개발하고 개선하는 역량을 
갖추도록 지원하는 것이다. 

AI 도구의 교육적 가치는 결국 그것을 설계한 교사의 전문성을 넘을 수 없다는 
원리에 따라, MAICE는 프롬프트와 평가도구 모두를 교사가 커스터마이징할 수 
있는 확장 가능한 연구 플랫폼으로 설계되었다.
```

**주요어 추가**: 교사 주도 연구, 프롬프팅 설계

### 2. 1장 서론 - 1.2.2 교육적 실천 목적 (신규 섹션)

```markdown
본 시스템은 "완성된 솔루션"이 아닌 "교사 주도 연구를 위한 확장 가능한 
플랫폼"으로 설계되었다. 이는 현재 에듀테크 산업에서 지배적인 
"상업적으로 주어진 완성된 도구를 소비만 하면 좋은 수업이 나온다"는 
접근에 대한 근본적 비판에서 출발한다.

AI 도구의 교육적 가치는 결국 그것을 설계하고 사용하는 교사의 전문성을 
넘을 수 없다. 마치 훌륭한 소설의 캐릭터가 작가의 능력을 넘지 못하듯, 
AI 교육 도구의 효과성은 그것을 설계한 교육적 철학과 교사의 맥락적 이해를 
초과할 수 없다.
```

**3가지 실천 목적**:
1. 학생의 반성적 사고 능력 탐구
2. 교사 주도 AI 교육 연구 플랫폼 제공
3. 프롬프팅 설계 연구의 지속가능성

- 학년별 난이도 조정

- **진화 사이클**: v1.0 → v1.1 → v2.0 (미적분) → v3.0 (확률)

**3-5. 교사 연구자 역량 강화 및 지속 가능한 연구 생태계** ⭐ (신규)
```
[설계] 프롬프트 + QAC 체크리스트 커스터마이징
[실험] 자신의 학급에서 실행
[수집] 세션 데이터 + QAC 평가 자동 수집
[분석] 효과성 분석
[개선] 데이터 기반 재설계
[검증] 신뢰도/타당도 재검증
[공유] 교사 커뮤니티와 공유
```

**핵심 차별점**:
- 프롬프트만 수정 (기존) → **프롬프트 + 평가도구 모두 수정** (MAICE)
- 단일 교사 연구 (기존) → **교사 커뮤니티 협업 연구** (MAICE)
- 주관적 평가 (기존) → **QAC 정량적 평가 + 공유** (MAICE)
