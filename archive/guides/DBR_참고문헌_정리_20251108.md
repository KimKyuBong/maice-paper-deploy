## 📚 **추가할 핵심 문헌 3편**

### **1. Collins, A., Joseph, D., & Bielaczyc, K. (2004)** ⭐⭐⭐

**서지 정보**:
- **제목**: Design research: Theoretical and methodological issues
- **저널**: *Journal of the Learning Sciences*, 13(1), 15-42
- **DOI**: 10.1207/s15327809jls1301_2
- **인용 횟수**: 6,000회 이상 (고전)

**핵심 내용**:
- DBR의 정의와 특징 정립
- 전통적 실험 연구와의 차이
- DBR의 5가지 핵심 특징:
  1. 이론 개발 목적
  2. 실제 맥락에서 수행
  3. 반복적 설계
  4. 방법론적 유연성
  5. 실용적 산출물

**논문 인용 시**:
```
본 연구는 설계 기반 연구(Design-Based Research, DBR) 방법론을 
채택하였다. DBR은 교육 이론을 실제 교육 맥락에서 검증하고, 
실용적 산출물을 개발하며, 반복적 개선을 통해 설계 원리를 
도출하는 연구 방법이다(Collins et al., 2004).
```

### **2. Wang, F., & Hannafin, M. J. (2005)** ⭐⭐⭐

**서지 정보**:
- **제목**: Design-based research and technology-enhanced learning environments
- **저널**: *Educational Technology Research and Development*, 53(4), 5-23
- **DOI**: 10.1007/BF02504682
- **인용 횟수**: 3,000회 이상

**핵심 내용**:
- DBR의 체계적 절차 제시
- DBR의 6가지 특징:
  1. Pragmatic (실용적)
  2. Grounded (이론 기반)
  3. Interactive (상호작용적)
  4. Iterative (반복적)
  5. Flexible (유연)
  6. Integrative (통합적)

**논문 인용 시**:
```
DBR은 실용성, 이론 기반, 반복성, 유연성, 통합성을 특징으로 하며
(Wang & Hannafin, 2005), 본 연구는 예비조사를 통한 문제 분석, 
이론 기반 설계, 현장 실험, 평가 및 개선의 반복적 과정을 거쳤다.
```

## 🎯 **논문 본문 수정 위치**

### **1. 연구 방법 섹션 (6장)**

추가할 내용:
```markdown
### 6.1 연구 방법론: Design-Based Research

본 연구는 설계 기반 연구(Design-Based Research, DBR) 방법론을 
채택하였다. DBR은 교육 이론을 실제 교육 맥락에서 검증하고, 
실용적 산출물을 개발하며, 반복적 개선을 통해 설계 원리를 
도출하는 연구 방법이다(Collins et al., 2004). Wang과 
Hannafin(2005)은 DBR의 핵심 특징으로 실용성(pragmatic), 
이론 기반(grounded), 반복성(iterative), 유연성(flexible), 
통합성(integrative)을 제시하였다.

본 연구의 DBR 과정은 다음과 같다:

**1) 분석 및 탐색 단계**
- 예비조사(2024년 5월): 학생 질문 385건 분석
- 문제 확인: 질문 품질의 구조적 문제 (72.3% 맥락 정보 부재)

**2) 설계 및 구축 단계**
- 이론 기반 설계: Dewey(1910)의 반성적 사고, Bloom의 교육목표분류학
- 시스템 개발: MAICE 멀티 에이전트 시스템 구현
- 평가 도구: QAC 루브릭 개발

**3) 평가 및 반성 단계**
- 현장 실험: 무작위 배정 A/B 테스트 (58명, 2024.10-11)
- 다각도 평가: AI 자동 채점 + 교사 평가
- 설계 원리 도출: 질문 명료화 중심 설계의 효과성 확인
```

## 🎯 **MAICE 연구의 DBR 적합성**

| DBR 특징 (Wang & Hannafin, 2005) | MAICE 연구 |
|----------------------------------|-----------|
| **Pragmatic** (실용적) | ✅ MAICE 시스템 (실제 사용 가능) |
| **Grounded** (이론 기반) | ✅ Dewey + Bloom 이론 |
| **Interactive** (상호작용) | ✅ 학생-시스템-교사 |
| **Iterative** (반복적) | ✅ 예비조사 → 본 실험 |
| **Flexible** (유연) | ✅ 교사 커스터마이징 |
| **Integrative** (통합적) | ✅ 양적+질적, 이론+실천 |
