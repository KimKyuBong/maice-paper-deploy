### 핵심 개념 상세

**① 템플릿과 도미노 모델**
- 3단계 구조: 베이스(base case), 가정(inductive hypothesis), 결론(inductive step)
- 도미노 비유: "첫 번째가 넘어지고, 하나가 넘어지면 다음도 넘어지면, 모두 넘어진다"
- 증명의 표준 형식 제공

**② 귀납가정 → 귀납결론 유도**
- k일 때 성립한 가정을 k+1 증명에 **어떻게 활용**하는가
- 논리적 연결고리 찾기
- 단순 형식 암기가 아닌 **논리적 연결 과정** 이해

**③ 등식 vs 부등식 전략**
- 등식: "필요한 재료만 딱 맞게" (정확성 강조)
- 부등식: "스페어 부품도 있는" (부등호 여유 활용)
- 명제 유형별 차별적 접근

## 🎯 반영의 학술적 의의

### 1. **실험 통제 (Experimental Control)** ✅

```
모든 학생이 동일한 사전 수업을 받음
         ↓
유일한 차이는 AI 모드 (Agent vs Freepass)
         ↓
실험 설계의 내적 타당도 확보
```

**중요성**:
- 실험 결과가 순수하게 AI 모드의 차이에서 비롯됨을 입증
- 외생 변수(교육 배경 차이) 통제

### 2. **재현 가능성 (Reproducibility)** ✅

```
구체적인 수업 내용 명시
         ↓
다른 연구자도 동일하게 재현 가능
         ↓
연구의 신뢰성 향상
```

**포함 내용**:
- ✅ 수업 구조 (선행 학습 + 해설)
- ✅ 강조한 핵심 개념 3가지
- ✅ 비유적 접근 방법
- ✅ 반복 강조 방식

### 3. **교육적 맥락 (Educational Context)** ✅

```
학생들의 배경 지식
         ↓
AI와의 대화 품질
         ↓
결과 해석의 기준선
```

**의미**:
- 학생들이 **어떤 기반**을 가지고 AI와 대화했는지 명확
- 질문-답변 품질의 해석 맥락 제공
- "공통 언어" 개념 도입으로 학습 효과 설명

### 4. **교사 주도성 (Teacher Agency)** ✅

```
교사가 설계한 비유 (도미노, 재료)
         ↓
매 수업 반복 강조
         ↓
학생들의 내면화
         ↓
AI 대화에서 자연스럽게 활용
```

**강조점**:
- 단순한 "AI 도구 사용" 연구가 아님
- **교사 주도 설계 + AI 도구**의 결합
- 교사의 교육적 전문성이 AI 활용에 중요함을 보여줌

### 5. **혼합 학습 모델 (Blended Learning)** ✅

```
[전통 수업] 개념 학습 + 공통 언어 형성
      ↓
[AI 활용] 심화 학습 + 개별화된 지원
      ↓
각각의 장점 극대화
```
