# 7장 재구성 계획

## 🎯 재구성의 목적

**문제**: 통계 중심 나열 → 핵심 메시지 상실
**목표**: 메시지 중심 서술 → 스토리 회복

---

## 📊 현재 구조의 문제점

### 문제 1: 절 수 과다 (10개)
```
1. 시스템 구축
2. 시스템 배포
3. 명료화 수행률
4. 명료화 유형
5. 우수 사례들
6. AI 평가
7. 모드별 비교
8. 하위권 효과
9. 교사 평가 ← 너무 김!
10. AI-교사 비교
```
→ 독자가 길을 잃음

### 문제 2: 주객 전도
```
주요 증거: AI 평가 (N=280) → 1개 절
보조 증거: 교사 예비 (N=100) → 2개 절 (9, 10)
```
→ 예비 연구가 과대 강조됨

### 문제 3: 중복과 산만함
```
명료화: 3개 절 (3, 4, 5)
AI 평가: 3개 절 (6, 7, 8)
교사 평가: 2개 절 (9, 10)
```
→ 같은 내용 반복

## 🆕 새로운 구조 (4개 절)

### 구조 설계 원칙
1. **메시지 중심**: 각 절이 명확한 메시지 전달
2. **증거 균형**: 주요(AI) vs 보조(교사) 비중 조정
3. **논리적 흐름**: 문제→해결→효과→검증

### 새로운 4개 절 구조

```
## 1. 연구 실행 및 데이터 수집 (~200줄)
   메시지: "시스템을 성공적으로 배포하고 280개 세션을 수집했다"
   
   1-가. 시스템 배포
   1-나. 데이터 수집 현황
   1-다. 명료화 프로세스 작동 확인
   
## 2. 명료화 모드의 학습 효과 (AI 평가) (~400줄) ⭐⭐⭐
   메시지: "명료화가 학습을 돕는다 (주요 증거)"
   
   2-가. 평가 도구 및 방법 (QAC 체크리스트)
   2-나. 평가자 간 신뢰도 (3개 AI 모델, α=0.840)
   2-다. 전체 효과: 학습 지원 우수 (p=0.034, d=0.275)
   2-라. 차별적 효과: 하위권 큰 효과 (+8.00, p=0.029, d=1.323)
   2-마. 누적 효과: 반복 사용 시 증가 (+0.99, d=0.298)
   
## 3. 교사 예비 평가를 통한 방향성 검증 (~300줄) ⭐
   메시지: "교사 평가도 같은 방향성을 보였다 (보조 검증)"
   
   3-가. 예비 연구 설계 (N=100, 2명)
   3-나. 평가자 간 신뢰도 (r=0.644)
   3-다. 전체 효과 (+2.25, p=0.031, d=0.307)
   3-라. 하위권 효과 (+6.91, p=0.009, d=1.117)
   3-마. AI 평가와의 일치 (Claude-4.5 r=0.771)
   3-바. 연구의 한계 및 후속 연구
   
## 4. 학생 인식 및 종합 (~200줄)
   메시지: "학생들도 긍정적으로 평가했다 (수렴)"
   
   4-가. 학생 설문 결과 (60% 선호)
   4-나. 삼각 검증: AI·교사·학생의 수렴
   4-다. 종합 및 시사점
```

**총 예상 길이**: ~1,100줄 (현재 2,182줄의 50%)

## 📝 각 절별 상세 계획

### 1절: 연구 실행 및 데이터 수집

**목적**: 연구가 제대로 수행되었음을 보여줌
**핵심**: 시스템 작동 + 데이터 품질

**포함 내용**:
- Docker 배포 성공
- 280개 유효 세션 (Agent 118, Freepass 162)
- 명료화 83.3% 작동
- 데이터 품질 확인

**제거 내용**:
- 너무 세세한 기술 스펙
- 우수 사례 긴 인용 (부록으로)

---

### 2절: 명료화 모드의 학습 효과 (주요 증거)

**목적**: AI 평가로 효과 입증 (N=280)
**핵심**: 학습 지원 우수 + 하위권 큰 효과

**포함 내용**:
- QAC 체크리스트 소개
- 3개 AI 모델, α=0.840
- **전체 효과**: C2 학습 지원 (p=0.034, d=0.275)
- **하위권 효과**: +8.00점 (p=0.029, d=1.323) ⭐⭐⭐
- **누적 효과**: +0.99점 (d=0.298)

**강조점**:
- 이것이 주요 증거 (N=280)
- 통계적으로 유의
- 하위권에 특히 효과적

---

### 3절: 교사 예비 평가 (보조 검증)

**목적**: 교사도 같은 결론 (방향성 일치)
**핵심**: 예비 연구 + 한계 명시

**포함 내용**:
- 예비 연구 설계 (N=100, 2명)
- 신뢰도 r=0.644
- 전체 +2.25 (p=0.031)
- 하위권 +6.91 (p=0.009)
- Claude-4.5와 유사 패턴
- **한계 명시**: 예비 연구, 표본 작음

**강조점**:
- 이것은 "보조 검증"
- AI 평가 결과를 "지지"
- 후속 연구 필요

---

### 4절: 학생 인식 및 종합

**목적**: 삼각 검증 + 종합 메시지
**핵심**: 3개 독립 증거의 수렴

**포함 내용**:
- 학생 설문 60% 선호
- 응답 편향 고려
- AI·교사·학생 수렴
- **핵심 메시지 3개**:
  1. 명료화가 학습을 돕는다
  2. 하위권에 특히 효과적
  3. 교육 격차 해소 가능성
