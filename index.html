    수학 학습에서 질문 명료화를 지원하는 AI에이전트 설계 및 개발: 고등학교 2학년 수학적 귀납법 단원 중심으로¶

    논문 정보

    • 저자: 김규봉
    • 소속: 부산대학교 AI융합교육전공
    • 학번: 202373389
    • 키워드: 질문 명료화, AI 에이전트, 반성적 사고, 수학적 귀납법, 멀티에이전트 시스템, Dewey, 설계 기반 연구, 교사 주도 연구, 프롬프팅 설계
    • 연구 대상: 고등학교 2학년 학생 58명
    • 참고문헌: 18편 (원문 검증 10편, 본문 인용 100%)
    • 마지막 업데이트: 2025년 11월 8일

    📄 논문 초록¶

    생성형 AI 도구의 교육 현장 도입이 증가하고 있으나, 학생 질문의 질적 문제가 효과적인 학습을 저해하는 주요 요인으로 확인되고 있다. 본 연구의 예비조사(385건의 질문 분석)에서 학생 질문의 72.3%가 학습 맥락 정보를 제공하지 않았으며, 45.8%는 질문 구조가 불명확하고, 45.5%는 수학적 전문성이 결여된 것으로 나타났다. 현재 대부분의 AI 교육 도구가 채택하고 있는 프리패스(FreePass) 방식—질문의 질과 무관하게 즉시 답변을 제공하는 방식—은 학생들이 스스로 질문을 구조화하고 사고를 확장하는 과정을 지원하지 못하여 깊이 있는 학습을 제한한다는 한계가 있다.

    본 연구는 이러한 문제를 해결하기 위해 Dewey의 반성적 사고 이론(1910)과 Bloom의 교육목표분류학(Anderson & Krathwohl, 2001)에 기반한 다중 에이전트 시스템 MAICE(Mathematical AI Chatbot for Education)를 설계·개발하였다. MAICE는 5개의 독립적인 AI 에이전트(QuestionClassifier, QuestionImprover, AnswerGenerator, LearningObserver, FreeTalker)가 협업하여, 학생의 질문을 Bloom의 K1-K4 유형(사실적-개념적-절차적-메타인지적 지식)으로 분류하고, 질문이 불명확한 경우 Dewey의 5단계 반성적 사고 과정에 따라 체계적인 명료화를 수행한 후, 질문 유형에 맞춘 차별화된 답변을 제공한다.

    연구 설계: 고등학교 2학년 58명을 대상으로 수학적 귀납법 단원에서 무작위 배정 A/B 테스트를 수행하였다(2024.10.21~11.1, 283개 세션).

    단일 세션 효과: Agent 모드(명료화 포함)는 단일 세션에서 학습 지원 점수가 Freepass 모드보다 통계적으로 유의하게 높았다(C3: p=0.034, d=0.282).

    누적 효과(AI 평가): 283개 세션 회귀 분석에서(AI 3모델 평균), Agent는 세션당 +0.29점 증가한 반면(p=0.096), Freepass는 +0.05점 증가하였다(p=0.641). 두 기울기 차이는 통계적으로 유의하지 않았다(p=0.253).

    학업 하위권 효과: 학업 하위권(객관식 하위 33%)에서 교사 평가(n=32)는 Agent가 세션당 +2.79점, Freepass가 -2.22점 변화하여, 기울기 차이가 통계적으로 유의하였다(+5.02점/세션, p=0.011). 실제 평균 점수 차이는 약 5점이었다. AI 모델 평가에서는 통계적으로 유의하지 않았다(p=0.245).

    질문 명료화의 중요성: 예비조사에서 질문-답변 품질 간 강한 정적 상관을 확인하였다(r=0.691, p<0.001).

    본 연구는 세 가지 층위에서 기여한다. 첫째, Dewey의 "문제 명료화" 단계가 학습 효과를 높이는 유효한 방법임을 A/B 테스트로 검증하였다(단일 효과 p=0.034). 둘째, AI 교육 도구 설계를 즉시 답변 중심에서 질문 명료화 중심으로 전환하고, 전체 283개 세션의 회귀 분석을 통해 Agent가 Freepass보다 더 빠른 학습 증가 추세를 보였다(통계적으로 유의하지 않음, p=0.253). 특히 학업 하위권에서 교사 평가를 통해 세션 반복에 따른 통계적으로 유의한 차이를 확인하여(p=0.011), 학업 하위권 학생 지원 가능성을 제시하였다. 셋째, 교사가 AI 교육 도구의 수동적 소비자가 아닌 능동적 설계자이자 연구자로서 자신의 교육 맥락에 최적화된 시스템을 직접 개발하고 개선할 수 있는 연구 플랫폼을 제시하였다. "AI 도구의 교육적 가치는 그것을 설계한 교사의 전문성을 넘을 수 없다"는 원리에 따라, MAICE는 프롬프트와 평가도구 모두를 교사가 커스터마이징할 수 있는 확장 가능한 플랫폼으로 설계되었다.

    주요 연구 결과¶

    핵심 발견

    1. 단일 세션 학습 지원 효과 ✅
      C3 학습 지원 항목에서 Agent 모드가 통계적으로 유의하게 우수하였다 (Agent 3.74 vs Freepass 3.34, one-tailed p=0.034, Cohen's d=0.282).

    2. 누적 학습 효과 (283개 세션 회귀 분석, 3개 AI 모델 평균)
      Agent는 세션당 +0.29점 증가 추세를 보인 반면(p=0.096, 한계적 유의), Freepass는 +0.05점의 약한 증가를 보였다(p=0.641). 두 기울기 차이는 통계적으로 유의하지 않았다(p=0.253).

    3. 학업 하위권에 통계적으로 유의한 효과 ⭐
      AI 모델 평가에서는 Agent가 +0.54점/세션, Freepass가 +0.09점/세션 증가하여 통계적으로 유의하지 않았으나(p=0.245), 교사 평가(n=32)에서는 Agent가 +2.79점/세션, Freepass가 -2.22점/세션의 변화를 보여 세션 반복에 따른 기울기 차이가 통계적으로 유의하였다 (p=0.011). 실제 관찰된 평균 점수 차이는 약 5점이었다.

    학술적 기여¶

    본 연구는 다음 세 가지 층위에서 기여한다:

    1. 교육학 이론의 실증적 검증: - Dewey의 "문제의 명료화" 단계가 학습 효과를 높이는 유효한 방법임을 A/B 테스트로 검증 - 질문-답변 상관관계(r=0.691) 발견으로 질문 품질의 중요성 실증 - 프리패스 방식의 제한점 확인 (반복 사용 시 증가 추세 약함)

    2. AI 교육 도구 설계의 대안적 접근: - 즉시 답변 중심 → 질문 명료화 중심 설계 전환 - 멀티 에이전트 구조로 Dewey 5단계 + Bloom K1-K4 체계적 구현 - 학업 하위권 학생에게 실용적 효과 (교사 평가 p=0.011) → 학업 하위권 학생 지원 가능성

    3. 교사 주도 AI 교육 연구 플랫폼 제시: - 상업적 완성품 소비 → 교사가 프롬프트와 평가도구를 직접 설계·개선하는 연구 플랫폼 - "AI 도구의 교육적 가치는 그것을 설계한 교사의 전문성을 넘을 수 없다"는 원리 제시 - 에듀테크 접근의 전환: 교사 = 수동적 소비자 → 능동적 설계자 겸 연구자

    📚 논문 구성¶

    • 1장. 서론

      연구의 필요성, 목적, 연구 문제, 용어 정의

      주요 내용: 학생 질문의 질적 문제, 프리패스 방식의 한계

    • 2장. 이론적 배경

      질문 생성 이론, Dewey의 반성적 사고, 멀티 에이전트 시스템

      주요 내용: 명료화의 교육학적 근거

    • 3장. MAICE 교육 시스템 아키텍처

      설계 철학, 3계층 아키텍처, 5개 에이전트 역할과 협업

      주요 내용: 명료화 중심 학습, Dewey/Bloom 이론 구현, A/B 테스트 설계

    • 4장. MAICE 시스템 구현

      기술 스택, 프롬프트 엔지니어링, 통신 메커니즘, 배포 전략

      주요 내용: Gemini API 연동, Redis Streams, Docker 배포

    • 5장. 수학적 귀납법 적용

      수학적 귀납법의 교육적 특성, 단원별 맞춤 설계

      주요 내용: 명료화 프로세스 실제 적용 사례

    • 6장. 연구 방법

      연구 방법론(DBR), 연구 대상, 도구, 자료 수집 및 분석

      주요 내용: Design-Based Research, 58명 A/B 테스트, QAC 루브릭

    • 7장. 결과

      실증 분석 결과, 통계 검정, 질문 품질 변화 분석

      주요 내용: 누적 학습 효과, 차별적 효과성

    • 8장. 논의 및 결론

      연구 결과 해석, 시사점, 제한점, 방법론적 의의, 후속 연구

      주요 내용: Dewey 이론 실증, DBR 방법론적 기여, 교육적 함의

    • 9장. 참고문헌
