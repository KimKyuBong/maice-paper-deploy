```
특히 하위권(Q1)에서 +6.91점의 큰 효과(p=0.009, d=1.117)를 보였으며, 
Anthropic AI가 이와 거의 동일한 패턴(+7.01점)을 감지하여

```

**수정 후**:
```

예비 연구에서 명료화 모드가 통계적으로 유의하게 높았다
(+2.25점, p=0.031, d=0.307). 제한적 표본(N=100)에도 불구하고

관찰되었으며, Anthropic AI가 유사한 패턴(+7.01점)을 보여 
평가 방향성이 일치하였다.
```
