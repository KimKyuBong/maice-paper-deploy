## 9. 핵심 권장 사항

### 🔴 즉시 수행 필요 (논문 신뢰성 확보를 위한 필수 사항)

1. **인간 평가자 검증** (최우선! ⚠️ 현재 미완료)
   - **문제**: 현재 ICC = 0.81 주장은 근거 없음 (실제 계산 결과 ICC = -0.06)
   - **필요**: 수학교사 1명 + 국어교사 1명 섭외
   - **방법**: 20-30개 세션 독립 채점 후 AI vs 인간 상관계수 분석
   - **목표**: Pearson r > 0.7
   - **소요 시간**: 3-5일
   - **상태**: ❌ 미실시

2. **거짓 내용 삭제 완료** ✅
   - ICC = 0.81 삭제
   - 루브릭 "타당도 검증" → "통계적 일관성 분석"으로 수정
   - 부록 G 제목 수정: "타당도 분석" → "AI 채점 일관성 분석"
   - 한계점 명시: 순환논리, 준거 타당도 미검증
