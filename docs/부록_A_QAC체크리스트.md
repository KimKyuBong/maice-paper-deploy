# 부록 A: QAC 체크리스트

## 1. 전체 구조

**QAC (Question-Answer-Context) 체크리스트**는 학생-AI 학습 세션의 교육적 효과성을 평가하기 위한 도구이다.

- **총 점수**: 40점 만점
- **구성**: 3개 영역, 8개 항목, 32개 체크리스트 요소
- **이론적 기반**: Dewey(1910) 반성적 사고, King(1994) 질문 생성 이론, Bloom(1956) 교육목표분류학

| 영역 | 항목 | 배점 | 체크리스트 요소 |
|------|------|:----:|:---------------:|
| **A. 질문** | A1. 수학적 전문성 | 5점 | 4개 |
| | A2. 질문 구조화 | 5점 | 4개 |
| | A3. 학습 맥락 | 5점 | 4개 |
| **B. 답변** | B1. 학습자 적합성 | 5점 | 4개 |
| | B2. 설명 체계성 | 5점 | 4개 |
| | B3. 학습 확장성 | 5점 | 4개 |
| **C. 맥락** | C1. 대화 일관성 | 5점 | 4개 |
| | C2. 학습 지원 | 5점 | 4개 |
| **합계** | **8개 항목** | **40점** | **32개 요소** |

**점수 산정 방식**:
```
각 항목 점수 = 충족 요소 개수 + 1
- 0개 충족: 1점
- 1개 충족: 2점
- 2개 충족: 3점
- 3개 충족: 4점
- 4개 충족: 5점 (만점)
```

## 2. A영역: 질문 평가 (15점)

### A1. 수학적 전문성 (5점)

| 코드 | 요소명 | 평가 기준 |
|------|--------|----------|
| A1-1 | 개념 정확성 | 수학 용어를 정확하게 사용했는가? |
| A1-2 | 교과과정 위계성 | 학년 수준에 맞는 개념인가? |
| A1-3 | 용어 적절성 | 전문 용어를 적절히 사용했는가? |
| A1-4 | 문제 방향 구체성 | 해결하려는 문제가 구체적인가? |

### A2. 질문 구조화 (5점)

| 코드 | 요소명 | 평가 기준 |
|------|--------|----------|
| A2-1 | 단일성 | 한 번에 하나의 명확한 질문을 하는가? |
| A2-2 | 조건 완전성 | 필요한 조건/정보를 모두 제시했는가? |
| A2-3 | 문장 논리성 | 문장이 논리적으로 구성되었는가? |
| A2-4 | 의도 명확성 | 무엇을 알고 싶은지 명확한가? |

### A3. 학습 맥락 (5점)

| 코드 | 요소명 | 평가 기준 |
|------|--------|----------|
| A3-1 | 현재 단계 설명 | 학년/단원/진도를 언급했는가? |
| A3-2 | 선행학습 언급 | 이전에 배운 내용을 언급했는가? |
| A3-3 | 어려움 구체화 | 어디서 막혔는지 구체적으로 말했는가? |
| A3-4 | 학습 목표 제시 | 무엇을 배우고 싶은지 목표를 제시했는가? |

## 3. B영역: 답변 평가 (15점)

|------|--------|----------|

| B2-2 | 단계별 논리 | 각 단계가 논리적으로 연결되는가? |

| 코드 | 요소명 | 평가 기준 |
|------|--------|----------|

| B3-3 | 오개념 교정 | 잘못된 이해를 바로잡았는가? |
| B3-4 | 자기주도 유도 | 스스로 탐구하도록 유도했는가? |

## 4. C영역: 맥락 평가 (10점)

|------|--------|----------|

## 5. 영역별 총점 계산

- **질문 영역 (A)**: A1 + A2 + A3 = 15점
- **답변 영역 (B)**: B1 + B2 + B3 = 15점
- **맥락 영역 (C)**: C1 + C2 = 10점
- **전체 총점**: A + B + C = 40점
