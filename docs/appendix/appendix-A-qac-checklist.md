# 부록 A. QAC 체크리스트 상세 설명서

## Question-Answer-Context Checklist 전체 평가 기준

---

## A. 질문 영역 (Question Domain) - 15점 만점

### A1. 수학적 전문성 (Mathematical Professionalism) - 5점

| 점수 | 기준 | 설명 |
|------|------|------|
| **5점** | 매우 우수 | 수학적 개념, 원리, 절차를 정확하게 사용하며, 수학 학습 목적에 완벽히 부합 |
| **4점** | 우수 | 수학적 용어를 대체로 정확히 사용하며, 일부 표현상 오류는 있으나 학습 맥락상 적절 |
| **3점** | 보통 | 기본적인 수학적 맥락은 있으나, 용어 사용이 부정확하거나 개념 이해가 불완전 |
| **2점** | 미흡 | 수학과 관련되어 있으나, 개념 오류가 많거나 용어 사용이 부적절 |
| **1점** | 매우 미흡 | 수학과 무관하거나, 수학적 맥락을 전혀 파악할 수 없음 |

**평가 세부 기준**:
- 수학 용어의 정확성 (예: "일반항" vs "명시적 공식")
- 수학적 개념의 올바른 사용
- 수학 학습 목적 부합도
- 교육과정 용어 일치성

---

### A2. 질문 구조화 (Question Structure) - 5점

| 점수 | 기준 | 설명 |
|------|------|------|
| **5점** | 매우 우수 | 질문 대상, 범위, 초점이 명확하며, 구체적 어려움을 정확히 제시 |
| **4점** | 우수 | 질문이 명확하나, 일부 세부사항이 불명확하거나 추가 설명 필요 |
| **3점** | 보통 | 질문 의도는 파악 가능하나, 구체성이 부족하거나 모호한 부분 존재 |
| **2점** | 미흡 | 질문 대상이나 범위가 불명확하여, 어떤 답을 원하는지 파악하기 어려움 |
| **1점** | 매우 미흡 | 질문 의도를 전혀 파악할 수 없거나, 단순 수식 제시만 있음 |

**평가 세부 기준**:
- 핵심 질문의 단일성 (한 번에 하나의 질문)
- 조건 제시의 완결성
- 문장 구조의 논리성
- 질문 의도의 명시성

---

### A3. 학습 맥락 적용 (Learning Context) - 5점

| 점수 | 기준 | 설명 |
|------|------|------|
| **5점** | 매우 우수 | 학습자 수준, 선수지식, 학습 목적, 이해 상태를 명확히 제시 |
| **4점** | 우수 | 학습 맥락 정보 중 3가지 이상 포함 |
| **3점** | 보통 | 학습 맥락 정보 중 2가지 포함 |
| **2점** | 미흡 | 학습 맥락 정보 중 1가지만 포함 |
| **1점** | 매우 미흡 | 학습 맥락 정보가 전혀 제시되지 않음 |

**평가 세부 기준**:
- 현재 학습 단계 명시 (예: "수학적 귀납법을 배우는 중")
- 선수 지식 연계 (예: "수열은 알지만")
- 구체적 어려움 제시 (예: "n=k+1 단계에서 막힘")
- 학습 목표 명시 (예: "증명 방법을 이해하고 싶음")

## B. 답변 영역 (Answer Domain) - 15점 만점

### B1. 학습자 맞춤도 (Learner Customization) - 5점

| 점수 | 기준 | 설명 |
|------|------|------|
| **5점** | 매우 우수 | 학습자 수준을 정확히 파악하고, 선수지식을 연계하며, 개인화된 피드백 제공 |
| **4점** | 우수 | 학습자 수준을 대체로 파악하고, 적절한 난이도로 설명 제공 |
| **3점** | 보통 | 일반적 수준의 설명으로, 특별한 맞춤화는 없으나 크게 부적절하지 않음 |
| **2점** | 미흡 | 학습자 수준과 맞지 않거나, 선수지식 연계 부족 |
| **1점** | 매우 미흡 | 학습자 수준을 전혀 고려하지 않거나, 완전히 잘못된 수준의 설명 |

**평가 세부 기준**:
- 학습자 수준별 접근 (초급/중급/고급)
- 선수지식 연계성 (이미 아는 것 활용)
- 학습 난이도 조절 (적절한 도전 수준)
- 개인화된 피드백 (질문의 맥락 반영)

---

### B2. 설명의 체계성 (Explanation Systematicity) - 5점

| 점수 | 기준 | 설명 |
|------|------|------|
| **5점** | 매우 우수 | 개념이 위계적으로 설명되고, 단계별 논리가 명확하며, 핵심이 강조됨 |
| **4점** | 우수 | 체계적 설명이나, 일부 논리 비약이나 순서 문제 있음 |
| **3점** | 보통 | 필요한 내용은 포함되나, 체계성이 다소 부족 |
| **2점** | 미흡 | 설명 순서가 혼란스럽거나, 핵심과 부수적 내용 구분 불명확 |
| **1점** | 매우 미흡 | 논리 체계가 없거나, 설명 구조를 파악할 수 없음 |

**평가 세부 기준**:
- 개념 설명의 위계화 (기초→심화)
- 단계별 논리 전개 (순차적 설명)
- 핵심 요소 강조 (중요한 것 표시)
- 예시 활용의 적절성 (이해를 돕는 예시)

---

### B3. 학습 내용 확장성 (Learning Expandability) - 5점

| 점수 | 기준 | 설명 |
|------|------|------|
| **5점** | 매우 우수 | 심화학습 방향 제시, 응용문제 연계, 오개념 교정, 자기주도 학습 유도 모두 포함 |
| **4점** | 우수 | 위 4가지 중 3가지 포함 |
| **3점** | 보통 | 위 4가지 중 2가지 포함 |
| **2점** | 미흡 | 위 4가지 중 1가지만 포함 |
| **1점** | 매우 미흡 | 단순 답변 제공만으로, 학습 확장 요소 전혀 없음 |

**평가 세부 기준**:
- 심화학습 방향 제시 (다음 단계 안내)
- 응용문제 연계성 (실제 문제 적용)
- 오개념 교정 전략 (잘못 이해한 부분 수정)
- 자기주도 학습 유도 (스스로 탐구하도록 유도)

## C. 대화 맥락 영역 (Context Domain) - 15점 만점

### C1. 명료화 효과성 (Clarification Effectiveness) - 5점

| 점수 | 기준 | 설명 |
|------|------|------|
| **5점** | 매우 우수 | 명료화 질문이 질문 개선에 결정적으로 기여하여, 모호한 질문이 구체화됨 |
| **4점** | 우수 | 명료화 과정이 효과적으로 진행되어, 질문 품질이 향상됨 |
| **3점** | 보통 | 명료화 시도는 있으나, 질문 개선 효과가 제한적 |
| **2점** | 미흡 | 명료화 질문이 부적절하거나, 질문 개선에 기여하지 못함 |
| **1점** | 매우 미흡 | 명료화가 전혀 수행되지 않거나, 오히려 혼란 초래 |
| **0점** | 해당 없음 | Freepass 모드 (명료화 기능 없음) |

**평가 세부 기준**:
- 명료화 질문의 적절성
- 학생 응답 유도 효과
- 질문 구체화 정도
- 명료화 성공률

---

### C2. 대화 일관성 (Dialogue Consistency) - 5점

| 점수 | 기준 | 설명 |
|------|------|------|
| **5점** | 매우 우수 | 학습 목표를 중심으로 일관된 대화가 유지되며, 이전 맥락을 완벽히 활용 |
| **4점** | 우수 | 대화가 대체로 일관되며, 이전 맥락을 적절히 활용 |
| **3점** | 보통 | 대화 흐름이 유지되나, 일부 맥락 단절이 있음 |
| **2점** | 미흡 | 대화가 자주 단절되거나, 이전 맥락을 무시 |
| **1점** | 매우 미흡 | 대화 일관성이 전혀 없거나, 주제가 계속 변경됨 |

**평가 세부 기준**:
- 학습 목표 중심 대화 유지
- 이전 대화 맥락 활용
- 주제 일관성
- 논리적 대화 흐름

---

### C3. 학습 과정 지원성 (Learning Support) - 5점

| 점수 | 기준 | 설명 |
|------|------|------|
| **5점** | 매우 우수 | 사고 과정 유도, 이해 점검, 메타인지 활동 촉진이 모두 탁월하게 수행됨 |
| **4점** | 우수 | 학습 과정 지원이 효과적으로 이루어지며, 대부분의 요소 포함 |
| **3점** | 보통 | 기본적인 학습 지원은 있으나, 깊이 있는 사고 유도는 제한적 |
| **2점** | 미흡 | 단순 정보 제공 수준으로, 학습 과정 지원이 부족 |
| **1점** | 매우 미흡 | 학습 지원이 거의 없거나, 오히려 학습을 방해 |

**평가 세부 기준**:
- 사고 과정 유도 (스스로 생각하게 만들기)
- 이해 점검 질문 (이해했는지 확인)
- 메타인지 활동 촉진 (자기 학습 점검)
- 비계 설정 (단계별 지원)

## 종합 점수 체계

**45점 만점 기준**:
- A 영역 (질문): 15점
- B 영역 (답변): 15점
- C 영역 (맥락): 15점

**15점 환산 기준**:
- 각 영역 최고 점수를 5점으로 환산
- 질문 5점 + 답변 5점 + 맥락 5점 = 총 15점

**품질 등급 분류**:
- **Excellent** (12-15점): 매우 우수한 학습 대화
- **Good** (8-11점): 양호한 학습 대화
- **Fair** (4-7점): 개선이 필요한 학습 대화
- **Poor** (0-3점): 학습 효과가 제한적인 대화

---

## 채점 시 주의사항

1. **독립적 평가**: 각 영역은 독립적으로 평가하며, 다른 영역의 점수에 영향받지 않음
2. **맥락 고려**: 학생의 학년, 학습 상황, 질문 배경을 고려하여 평가
3. **교육적 가치 중심**: 단순히 정보 정확성이 아닌, 교육적 효과를 중심으로 평가
4. **일관성 유지**: 동일 평가자가 모든 세션을 평가하여 일관성 확보
