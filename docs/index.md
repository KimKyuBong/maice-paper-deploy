## 📄 논문 초록

본 연구는 고등학교 2학년 학생 53명을 대상으로 수학적 귀납법 학습에서 질문 명료화를 지원하는 AI agent 시스템 MAICE의 효과를 검증하였다. 기존 LLM의 즉시 답변 방식(Freepass)은 학생들의 모호한 질문을 그대로 수용하여 교육적 효과가 제한적이었다. 이에 본 연구는 Dewey의 반성적 사고 이론에 기반한 명료화 프로세스를 설계하고, 무작위 배정 A/B 테스트를 통해 179개 세션을 비교 분석하였다.

### 주요 연구 결과

!!! success "핵심 발견"
    1. **단일 세션 학습 지원 효과**  
       C3 학습 지원 항목에서 Agent 모드가 통계적으로 유의하게 우수하였다 (Agent 3.74 vs Freepass 3.34, one-tailed **p=0.034**, Cohen's d=0.282).
    
    2. **누적 학습 효과**  
       Agent 사용자는 세션 반복 시 평균 +0.63점 증가한 반면, Freepass 사용자는 -0.36점 감소하였다 (차이 +0.99점, Cohen's d=0.298). 특히 질문 능력이 크게 향상되었다 (Agent +2.18 vs Freepass +0.52).
    
    3. **학업 하위권에 통계적으로 유의한 효과** ⭐  
       객관식 하위 33% 학생에서 Agent가 압도적 우세를 보였다 (Agent +0.91 vs Freepass -0.70, **p=0.040**, **Cohen's d=1.204**, 매우 큰 효과).

### 학술적 기여

본 연구는 **A/B 테스트를 통해 명료화 프로세스와 학습 효과 간 인과관계를 확립**하였으며, Dewey의 "문제의 명료화" 단계가 실제 학습 효과를 높이는 핵심 요소임을 실증하였다. 특히 학업 하위권 학생들에게 통계적으로 유의한 효과를 보여, AI 기술이 교육 격차 해소에 기여할 수 있는 구체적 경로를 제시하였다. 또한 즉시 답변 제공 방식의 근본적 한계(반복 사용 시 점수 감소)를 실증적으로 밝혔다.
