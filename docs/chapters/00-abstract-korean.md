# 국문 초록

## 수학 학습에서 질문 명료화를 지원하는 AI에이전트 설계 및 개발
### - 고등학교 2학년 수학적 귀납법 단원 중심으로

---

## 요약

생성형 AI의 교육 현장 확산에도 불구하고 학생 질문의 낮은 품질이 효과적 학습을 저해하고 있다. 예비조사(385건, III장 1절 참조)에서 학생 질문의 72.3%가 학습 맥락 정보를 제공하지 않았으며, 대부분의 AI 도구가 채택한 즉시 답변 방식은 학생의 사고 과정 확장을 지원하지 못하는 한계를 보였다.

본 연구는 Dewey의 반성적 사고 이론을 기반으로 질문 명료화 프로세스를 핵심으로 하는 AI 에이전트 시스템 MAICE를 설계·개발하고, 고등학교 2학년 수학적 귀납법 단원에 적용하여 학습 효과를 검증하였다. 교사는 도미노 모델, 귀납 과정 연결, 등식/부등식 전략 등의 핵심 개념을 매 수업마다 반복 강조하여 학습 맥락을 제공하였다.

**연구 방법**: 고등학교 2학년 58명을 명료화 우선 모드(n=28)와 즉시 답변 모드(n=30)에 무작위 배정하여 3주간 A/B 테스트를 수행하였다(284개 유효 세션). 시스템은 Gemini 2.5 Flash를 사용하여 운영되었으며, 평가 방법의 한계를 상호 보완하기 위해 LLM 평가(N=284)와 교사 평가(N=100)를 병행하였다. 질문-답변-맥락 평가 체크리스트를 개발하여 3개 독립 AI 모델(Gemini 2.5 Flash, Claude 4.5 Haiku, GPT-5 mini)이 대규모 패턴 탐색을, 외부 수학 교사 2명이 교육적 타당성 검증을 수행하였다. 평가자 간 신뢰도는 LLM 간 α=0.872, 교사 간 ICC=0.716, LLM-교사 간 r=0.754(3개 모델 평균)로 양호하였다.

**주요 결과**: LLM-교사 이중 평가와 질적 분석(DB 로그 1,589건)을 통해 명료화 효과를 검증하였다. LLM 평가(N=284, 3개 모델 평균)에서 명료화 모드는 학습 지원에서 유의하게 우수하였고(C2: +0.28점, p=0.004, d=0.353), 특히 하위권 학생에게 큰 효과를 보였다(Q1 C2: p<0.001, d=0.855). 교사 평가(N=100)에서도 동일한 방향성이 관찰되었으며, 응답 영역에서 유의한 차이를 보였고(+1.28점, p=0.017, d=0.488), 하위권에서 매우 큰 효과(Q1 전체: +6.32점, p=0.013, d=0.992)가 일치하였다. LLM-교사 간 높은 상관(r=0.754)으로 상호 보완적 평가가 가능함을 확인하였다. 질적 분석(1,589건 대화 로그) 대상 고득점 세션 6개(32-34점)는 모두 Bloom 고차원 지식 차원(K2-K4) 포함을 보였으며, 특히 세션 156 (Q1 하위권, 34.33점, 1등)은 지식 차원 3단계 상승(K2→K3→K4)과 Dewey 5단계 시도를 보여 하위권 효과의 질적 메커니즘을 확인하였다. LLM 평가점수는 교사 평가(r=0.754), Bloom/Dewey 이론 정합성, 학생 자기 평가 수렴으로 교육적 타당성이 지지되었다. 학생 설문(N=47)에서 AI 상호작용 품질 4.37/5.0, 개념 이해 4.39/5.0, 시스템 만족도 4.62/5.0으로 높은 만족도를 보였으며, A/B 선호도 분석(N=44)에서 68.4%가 명료화 방식을 선호하였다.

**결론**: 질문 명료화 프로세스는 학습 지원, 특히 하위권 학생에 대한 단기적 효과를 확인하였다. 본 연구는 특정 맥락(고2, 수학적 귀납법, SW 특화고)의 탐색적 증거로서, Dewey 이론의 AI 구현 가능성을 보여주었다. LLM-교사 이중 평가의 상호 보완 모델을 제시하여 대규모 패턴 탐색과 전문가 검증을 조합하였으며, 교사 협력 기반의 확장 가능한 연구 플랫폼을 제시하였다. 다양한 맥락에서의 재현 연구가 필요하다.
