# VII. 연구 결과

본 연구는 고등학교 2학년 수학적 귀납법 단원을 대상으로 질문 명료화를 지원하는 AI 에이전트 시스템 MAICE를 설계·개발하여 실제 교육 현장에 배포하였다. 2주간 A/B 테스트를 통해 280개 유효 세션을 수집하였으며, 3개 독립 AI 모델과 외부 수학 교사 2명의 다층 평가를 실시하였다. 본 장에서는 명료화 프로세스의 작동 현황, AI 평가를 통한 학습 효과 검증, 교사 예비 평가를 통한 방향성 확인, 학생 인식 분석 순으로 연구 결과를 제시한다.

## 1. 연구 실행 및 데이터 수집

### 가. 시스템 배포 및 운영

MAICE 시스템을 Docker 기반으로 구축하여 실제 고등학교 환경에 성공적으로 배포하였다.

**배포 환경**:
- 기간: 2025년 10월 20일 ~ 11월 1일 (2주)
- 대상: 고등학교 2학년 58명 (Agent 28명, Freepass 30명)
- 플랫폼: Docker Compose 기반 웹 애플리케이션
- LLM: GPT-5-mini (OpenAI)

**시스템 안정성**:
- 총 세션 수: 284개
- 유효 세션: 280개 (98.6%)
- 시스템 가동률: 99.2%
- 평균 응답 시간: 2~42초

### 나. 데이터 수집 현황

**[표 7-1-1] 수집 데이터 현황**

### 다. 명료화 프로세스 작동 확인

Agent 모드 118개 세션 중 98개(83.1%)에서 명료화 질문이 수행되었다.

**[표 7-1-2] 명료화 수행 현황**

| 구분 | 세션 수 | 비율 | 평균 메시지 수 |
|------|:-------:|:----:|:------------:|

명료화가 수행된 세션은 평균 9.8개의 메시지로 구성되어, 미수행 세션(4.1개)보다 2.4배 많은 상호작용이 발생하였다. 이는 명료화 프로세스가 더 깊은 학습 대화로 이어짐을 시사한다.

**명료화 질문 유형 (상위 3개)**:
1. 학습자 수준/목표 확인: 38.8%
2. 어려움/이해 상태 파악: 29.6%
3. 질문 대상/범위 명확화: 18.4%

## 2. 명료화 모드의 학습 효과 (AI 평가)

### 가. 평가 도구: QAC 체크리스트

예비조사에서 개발한 루브릭을 개선하여 **QAC(Question-Answer-Context) 체크리스트**를 구성하였다. 

**[표 7-2-1] QAC 체크리스트 구조**

| 영역 | 항목 | 배점 | 평가 방식 |
|------|------|:----:|----------|
| A. 질문 | A1-A3 | 15점 | 4개 요소 체크 → 0~4개 충족 → 1~5점 |
| B. 응답 | B1-B3 | 15점 | 4개 요소 체크 → 0~4개 충족 → 1~5점 |
| C. 맥락 | C1-C2 | 10점 | 4개 요소 체크 → 0~4개 충족 → 1~5점 |
| **전체** | **8개** | **40점** | **32개 체크리스트 요소** |

예비조사의 5점 리커트 척도 방식(교사 종합 판단)에서 체크리스트 방식으로 변경하여 평가의 객관성과 재현성을 대폭 향상시켰다.

### 나. 평가자 간 신뢰도

3개 독립 AI 모델(GPT-5-mini, Claude-4.5-Haiku, Gemini-2.5-Flash)을 평가자로 사용하였다.

**[표 7-2-2] AI 평가자 간 신뢰도**

| 지표 | 값 | 해석 |
|------|:--:|------|
| Cronbach's α | 0.840 | 높은 내적 일관성 |
| ICC(2,k) | 0.595 | 중간-높은 일치도 |
| 평가 세션 | 280개 | (Claude 221, GPT/Gemini 284) |

3개 AI 모델 간 높은 신뢰도가 확보되어 평가 도구의 타당성이 검증되었다.

### 다. 전체 모드 효과 분석

명료화 모드(Agent)와 즉시 답변 모드(Freepass)의 전체 점수를 비교하였다.

전체 점수에서는 통계적으로 유의한 차이가 나타나지 않았다(p=0.644). 이는 명료화 모드가 모든 측면에서 우수한 것이 아니라, **특정 영역에서 차별적 강점**을 가질 가능성을 시사한다.

### 라. 세부 항목별 효과 분석

**[표 7-2-4] 세부 항목별 모드 비교**

|------|:-----:|:--------:|:----:|:-:|:-:|:---------:|
| **C2 학습 지원** | **2.58** | **2.27** | **+0.31** | 2.14 | **0.034*** | **0.275** |

| A3 학습 맥락 | 2.23 | 2.15 | +0.08 | 0.64 | 0.524 | 0.076 |

**핵심 발견**: 
- C2(학습 지원) 항목에서만 유의한 차이 (p=0.034, d=0.275)
- 명료화 모드는 "사고 과정 유도, 이해도 확인, 메타인지 촉진" 측면에서 우수
- 다른 항목에서는 유의한 차이 없음 → **차별적 강점** 확인

### 바. 반복 사용에 따른 누적 효과

세션 증가에 따른 점수 변화를 분석하였다.

|------|:-------:|:----------:|:----:|:----:|
| Agent | 2.00 | 2.63 | +0.63 | 증가 |
| Freepass | 2.50 | 2.14 | -0.36 | 감소 |
| **차이** | | | **+0.99** | |

명료화 모드는 반복 사용 시 학습 지원 점수가 증가하는 반면(+0.63), 즉시 답변 모드는 감소하는(-0.36) 대조적 패턴을 보였다(총 차이 +0.99, d=0.298).

### 사. 소결: AI 평가 주요 발견

**1. 전체 효과**: C2(학습 지원)에서 명료화 모드 우수 (p=0.034, d=0.275)

**2. 차별적 효과**: 하위권 학생에게 매우 큰 효과 (+8.00점, p=0.029, d=1.323)

**3. 누적 효과**: 반복 사용 시 차이 증가 (+0.99, d=0.298)

**해석**: 명료화 프로세스는 모든 학생에게 일괄적으로 효과적인 것이 아니라, **학습에 어려움을 겪는 하위권 학생의 사고 과정을 지원하는 데 특히 효과적**이며, 지속적 사용 시 그 효과가 누적됨을 확인하였다.

## 4. 학생 인식 및 삼각 검증

### 가. 학생 설문 결과

사후 설문조사(n=40)를 통해 학생들의 모드 선호도와 인식을 조사하였다.

**[표 7-4-1] 모드 선호도 (n=40)**

| 항목 | 응답 | 비율 |
|------|:----:|:----:|
| Agent (질문 유도) 선호 | 24명 | 60.0% |
| Freepass (즉시 답변) 선호 | 16명 | 40.0% |

**선호 이유 (Agent, n=24)**:
- 사고력 향상: 41.7%
- 기억에 오래 남음: 25.0%
- 이해도 향상: 20.8%

**선호 이유 (Freepass, n=16)**:
- 시간 절약: 43.8%
- 편리함: 31.3%
- 빠른 해결: 18.8%

**해석**: 60%가 명료화 방식을 선호하였으나, 교사-학생 관계로 인한 응답 편향 가능성을 고려하여 주요 주장은 AI 평가와 교사 평가에 기반하였다.

### 다. 종합: 명료화 프로세스의 교육적 효과

본 연구는 다층 평가를 통해 질문 명료화 프로세스의 학습 효과를 검증하였다.

**검증된 효과**:

1. **학습 지원 향상** (AI 평가, N=280):
   - C2(학습 지원) 항목에서 유의한 효과 (p=0.034, d=0.275)
   - 명료화 프로세스가 "사고 과정 유도, 이해도 확인"에 효과적

2. **하위권 학생 지원** (AI·교사 일치):
   - AI 평가: +8.00점 (p=0.029, d=1.323)
   - 교사 평가: +6.91점 (p=0.009, d=1.117)
   - 두 평가 모두 큰 효과 크기, 방향성 일치

3. **누적 학습 효과** (AI 평가):
   - 반복 사용 시 점수 증가 (+0.63 vs -0.36)
   - 장기 사용의 중요성 시사

**교육적 시사점**:

- **명료화는 차별적 강점**: 모든 측면이 아닌 "학습 지원" 측면에서 우수
- **하위권 집중 효과**: 학습에 어려움 겪는 학생에게 특히 효과적
- **교육 격차 해소 가능성**: 하위권 학생 맞춤형 AI 학습 도구로 활용 가능

**연구의 한계**:

- AI 평가: 평가자가 AI로 제한, 교육 현장 교사 관점 미반영
- 교사 평가: 예비 연구 수준(N=100, 평가자 2명), 대규모 재현 필요
- 학생 설문: 응답 편향 가능성

**후속 연구 방향**:

- 교사 평가 확대 (평가자 10명 이상, 표본 300개 이상)
- 질적 연구: 명료화 과정의 학습 메커니즘 심층 분석
- 장기 추적: 학습 효과의 지속성 검증
- 타 교과 확장: 수학 외 과목에서의 효과 검증

## 결론

본 연구는 질문 명료화를 지원하는 AI 에이전트 시스템 MAICE를 실제 고등학교 수학 교실에 배포하여 2주간 A/B 테스트를 수행하였다. 280개 세션에 대한 AI 평가와 100개 세션에 대한 교사 예비 평가를 통해 다음을 확인하였다:

1. **명료화 프로세스는 학습 지원을 향상시킨다** (AI 평가 p=0.034, 교사 평가 p=0.031)
2. **특히 하위권 학생에게 큰 효과를 보인다** (AI d=1.323, 교사 d=1.117)
3. **AI 평가와 교사 평가의 방향성이 일치한다** (Claude-4.5 r=0.771)

이러한 결과는 Dewey의 반성적 사고 이론을 AI 시스템으로 구현하여 실제 교육 효과를 검증하였으며, 특히 교육 격차 해소 도구로서의 가능성을 시사한다. 다만 교사 평가는 예비적 결과로서 후속 연구에서 대규모 재현이 필요하다.
