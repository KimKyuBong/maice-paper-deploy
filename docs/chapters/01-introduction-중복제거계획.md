# 1장 서론 중복 제거 계획

## 문제 진단

### 중복 1: [표 1-3] vs 본문 (87-116줄)

**[표 1-3] 학생 질문 문제 유형별 분석** (77-83줄)
- 맥락 없는 질문: 72.3%, 184건
- 모호한 질문: 45.8%, 25건  
- 수학 무관 질문: 45.5%, 26건

**중복 본문**:
- 87줄: "#### 1) 맥락 없는 개념 설명 요청 (학습맥락 1점: 72.3%, 184건)"
- 96줄: "#### 2) 모호하고 불명확한 질문 (질문구조 1점: 45.8%, 25건)"
- 107줄: "#### 3) 수학과 무관한 질문 (수학적전문성 1점: 45.5%, 26건)"

→ **표의 통계를 본문에서 그대로 반복!**

### 중복 2: [표 1-4] vs 본문 (142-184줄)

**[표 1-4] AI 답변 문제 유형별 분석** (132-138줄)
- 학습 확장 부족: 평균 1.832, 48.9%
- 맞춤화 실패: 평균 2.474, 27.6%
- 체계성 부족: 평균 2.765, 23.9%

**중복 본문**:
- 142줄: "#### 1) 학습자 맞춤도 문제 (평균 2.474점, 최저점 27.6%)"
- 159줄: "#### 3) 설명의 체계성 문제 (평균 2.765점, 최저점 23.9%)"
- 168줄: "#### 4) 학습 내용 확장성 문제 (평균 1.832점, 최저점 48.9%)"

→ **표의 통계를 본문에서 그대로 반복!**

## 수정 방안

### 원칙
1. **표는 그대로 유지** (요약 기능)
2. **본문은 표를 참조하되, 중복 통계 제거**
3. **실제 사례만 간결하게 제시** (표에 없는 정성적 내용)
4. **표 → 간단한 요약문 → 대표 사례 1~2개**

### 수정 전략

#### 표 1-3 다음 (현재 87-116줄 → 축소)

**현재 (30줄)**:
```
#### 1) 맥락 없는 개념 설명 요청 (학습맥락 1점: 72.3%, 184건)

학생들은 "지수의 확장을 알려줘", "log 실생활 활용 예시"... [긴 설명]

**실제 사례 (ID 294):** "근데 어떻게 증명한거야?"
[8줄 설명]
```

**수정 후 (10줄)**:
```
[표 1-3]에서 확인할 수 있듯이, 학생 질문에서 가장 심각한 문제는 **학습 맥락 부재(72.3%)**였다. 대표적 사례는 다음과 같다:

**실제 사례 (ID 294):** "근데 어떻게 증명한거야?"
- 어떤 증명인지, 어디가 어려운지, 학습자 수준이 무엇인지 전혀 파악 불가
- AI도 맥락을 알 수 없어 추가 정보 요청만 가능
```

#### 표 1-4 다음 (현재 140-184줄 → 대폭 축소)

**현재 (45줄)**:
```
#### 1) 학습자 맞춤도 문제: ... (평균 2.474점, 최저점 27.6%)

루브릭의 '학습자 맞춤도' 영역은... [긴 설명]

**실제 사례 (ID 130):** ...
[15줄 설명]

#### 2) 학습자 맞춤도 문제: 학습 난이도 조절 실패
[8줄]

#### 3) 설명의 체계성 문제: ... (평균 2.765점, 최저점 23.9%)
[8줄]

#### 4) 학습 내용 확장성 문제: ... (평균 1.832점, 최저점 48.9%)
[15줄]
```

**수정 후 (20줄)**:
```
[표 1-4]에서 확인할 수 있듯이, AI 답변에서 **학습 확장성 결여(평균 1.832점, 48.9% 최저점)**가 가장 심각한 문제였으며, 맞춤화 실패와 체계성 부족도 두드러졌다.

**맥락 오해 사례 (ID 130):** "지수의 확장에 대해 설명해줘"
→ AI가 수학 "지수(exponent)"를 비즈니스 "지점 확장"으로 오해하여 완전히 무관한 답변 제공

**수준 불일치 사례 (ID 158):** 고1 학생이 "로지스틱 회귀" 질문
→ AI가 대학 통계학 수준의 "odds ratio", "최대우도추정" 개념 사용

**교육과정 용어 부적절 (ID 202):** "수열이 뭐야?"
→ AI가 한국 표준 용어 "일반항" 대신 "명시적 공식(Explicit Formula)" 사용
→ 교사-학생 간 의사소통 혼란 초래
```
