# QAC 체크리스트 vs 학생 설문지: 역할 분담 분석

## 📊 핵심 원칙

**설문지는 QAC로 측정할 수 없는 부분을 보완하는 역할**

## 1. QAC 체크리스트의 측정 범위와 한계

### QAC가 측정하는 것 (객관적 데이터)

| 영역 | 측정 내용 | 데이터 소스 | 한계 |
|------|----------|------------|------|
| **A1-A3** | 질문 품질 (전문성, 구조, 맥락) | 세션 로그 (최종 질문) | ✅ 객관적 품질만 측정 |
| **B1-B3** | AI 답변 품질 (맞춤도, 체계성, 확장성) | 세션 로그 (AI 답변) | ✅ 객관적 품질만 측정 |
| **C1-C2** | 대화 맥락 (일관성, 지원성) | 세션 로그 (전체 대화) | ✅ 객관적 대화 흐름만 측정 |

### QAC로 알 수 없는 것 (주관적 경험) ❌

| 측정 불가능 항목 | 이유 | 설문지 필요성 |
|---------------|------|--------------|
| **1. 학생의 주관적 경험** | 대화 로그에는 느낌/감정이 기록되지 않음 | ⭐⭐⭐ 필수 |
| **2. 학습 방식 변화 인식** | 시간에 따른 변화는 세션 간 비교로만 추정 가능 | ⭐⭐⭐ 필수 |
| **3. 명료화 프로세스 태도** | "귀찮다" vs "도움된다" 등 태도는 로그에 없음 | ⭐⭐⭐ 필수 |
| **4. 메타인지적 자기 인식** | "내가 무엇을 모르는지 안다"는 자기 평가 불가 | ⭐⭐⭐ 필수 |
| **5. 만족도와 선호도** | AI가 평가할 수 없는 주관적 가치 판단 | ⭐⭐ 중요 |
| **6. 수학 학습 성과** | QAC는 대화 품질만 측정, 학습 결과는 별개 | ⭐⭐ 중요 |
| **7. 구체적 에피소드** | 로그는 있지만 맥락과 의미는 학생만 알 수 있음 | ⭐⭐ 중요 |
| **8. 미래 사용 의향** | 로그로 예측 불가능 | ⭐ 참고 |

## 2. 현재 설문지의 QAC 보완 적합성 검토

### ✅ 적합한 문항 (QAC와 중복 없음)

#### B1-B5: 대화 품질 인식 (주관적 경험)

| 문항 | QAC 측정 | 설문 측정 | 역할 분담 |
|------|---------|----------|----------|
| B1: 무엇을 모르는지 명확해짐 | ❌ (객관적 질문 품질만) | ✅ **학생의 주관적 느낌** | 보완 관계 ✅ |
| B2: 스스로 해결 방법 찾음 | ❌ (답변 품질만) | ✅ **자기주도 학습 경험** | 보완 관계 ✅ |
| B3: 다음 질문 알게 됨 | ❌ (대화 품질만) | ✅ **학습 방향성 인식** | 보완 관계 ✅ |
| B4: AI가 의도 이해 | C1 (대화 일관성)과 유사 | ✅ **학생 관점 평가** | **약간 중복, 하지만 관점 다름** ⚠️ |
| B5: 충분한 도움 | B3 (학습확장성)과 유사 | ✅ **전반적 만족도** | **약간 중복, 하지만 주관적** ⚠️ |

**분석**: B1-B3은 완벽한 보완 관계. B4-B5는 약간 중복되지만 QAC는 객관적 품질, 설문은 주관적 인식이므로 의미 있음.

#### C6-C9: 질문 능력 & 메타인지 (자기 인식) ⭐⭐⭐

| 문항 | QAC 측정 | 설문 측정 | 역할 분담 |
|------|---------|----------|----------|
| C6: 질문 명확하게 표현 | A2 (질문 구조)와 유사 | ✅ **자기 평가** | **QAC: 객관적 품질 vs 설문: 자기 인식** ✅ |
| C7: 맥락 정보 제시 ⭐ | A3 (학습 맥락)과 유사 | ✅ **습관 변화 인식** | **QAC로 알 수 없는 자기 인식** ✅✅✅ |
| C8: 무엇을 모르는지 파악 | ❌ | ✅ **메타인지** | 완전 보완 ✅✅✅ |
| C9: 스스로 질문 만듦 | ❌ | ✅ **자기주도 학습** | 완전 보완 ✅✅✅ |

**분석**: C 영역은 QAC의 핵심 한계(객관적 품질만 측정)를 완벽하게 보완. 특히 C7은 예비조사의 72.3% 문제를 학생 관점에서 확인.

#### D10-D12: 수학적 귀납법 이해도 ⭐⭐⭐

| 문항 | QAC 측정 | 설문 측정 | 역할 분담 |
|------|---------|----------|----------|
| D10-D12: 수학 이해도 | ❌ (대화 품질만) | ✅ **학습 성과** | 완전 보완 ✅✅✅ |

**분석**: QAC는 대화 과정만 측정. 학습 결과는 설문으로만 측정 가능.

#### E13-E15: 시스템 만족도 ⭐⭐

| 문항 | QAC 측정 | 설문 측정 | 역할 분담 |
|------|---------|----------|----------|
| E13-E15: 만족도/사용성 | ❌ | ✅ **주관적 평가** | 완전 보완 ✅✅ |

**분석**: TAM 기반 만족도는 QAC로 측정 불가능.

#### Part 1-5: 정성 문항 ⭐⭐⭐ (가장 중요)

| Part | QAC 측정 | 설문 측정 | 역할 분담 |
|------|---------|----------|----------|
| Part 1: AI 대화 경험 | ❌ | ✅ **구체적 에피소드, 시간 변화** | 완전 보완 ✅✅✅ |
| Part 2: 학습 방식 변화 | ❌ | ✅ **습관 변화 자각** | 완전 보완 ✅✅✅ |
| Part 3: 선호도 | ❌ | ✅ **가치 판단, 이유** | 완전 보완 ✅✅✅ |
| Part 4: 인상 깊은 순간 | ❌ | ✅ **의미 있는 경험** | 완전 보완 ✅✅✅ |
| Part 5: 개선 제안 | ❌ | ✅ **사용자 관점 피드백** | 완전 보완 ✅✅✅ |

**분석**: 정성 문항은 QAC로 절대 알 수 없는 부분을 완벽하게 포착.

### QAC 체크리스트의 역할 (RQ1 중심)

```
┌─────────────────────────────────────┐
│  QAC: 객관적 대화 품질 측정        │

├─────────────────────────────────────┤
│  ✅ 질문 품질 (A1-A3)              │

│  ✅ 대화 맥락 (C1-C2)              │
├─────────────────────────────────────┤
│  📊 RQ1 검증: 질문 품질 개선       │
│  - 학습맥락 72.3% 문제 해결 확인   │
│  - 질문 구조화 개선 확인           │
```

## 4. QAC와 설문지의 통합 분석 전략

### 4.1 정량적 통합 (Triangulation)

| 측정 대상 | QAC | 설문 | 통합 분석 |
|----------|-----|------|----------|
| **질문 맥락 제공** | A3 점수 (객관적) | C7 점수 (주관적) | 상관 분석: A3 ↑ → C7 ↑ 예상 |
| **대화 품질** | B, C 점수 (객관적) | B1-B5 점수 (주관적) | 상관 분석: 객관적 품질과 주관적 만족도 관계 |
| **학습 지원** | C2 점수 (객관적) | B2, C8-C9 (주관적) | Agent 그룹에서 강한 상관 예상 |

### 4.2 정성적 통합

```
[QAC 정량 결과] + [설문 정성 응답] = 완전한 이해

예시:
- QAC: Agent 그룹의 A3(학습맥락) 점수 4.2점 > Freepass 3.1점
- 설문 C7: "질문할 때 내 상황이나 배경 정보를 함께 제시한다" 
  → Agent 4.0점 > Freepass 3.3점 (자기 인식도 높음)
- 설문 Part 2: "질문을 더 구체적으로 하게 됐다" (Agent 그룹 정성 응답)

→ 종합: 명료화 프로세스가 객관적으로 질문 품질을 개선했고,
        학생도 이를 자각하며, 습관으로 내재화됨을 확인
```

## 6. 결론 및 권고사항

### ✅ 현재 설문지의 QAC 보완 적합성: 매우 우수

**총 20문항 분석 결과:**
- **15문항 (75%)**: QAC로 측정 불가능한 영역을 완벽하게 보완 ⭐⭐⭐
- **5문항 (25%)**: QAC와 약간 중복되나 주관적 관점으로 의미 있음 ✅

### 핵심 강점

1. **정량-정성 균형**: QAC(정량) + 설문(정량+정성) = 완전한 데이터
2. **객관-주관 보완**: QAC(객관적 품질) + 설문(주관적 인식) = 삼각 검증
3. **과정-결과 측정**: QAC(대화 과정) + 설문(학습 결과) = 인과 관계 추론
4. **정성 데이터 강화**: Part 1-5의 가이드 질문으로 풍부한 에피소드 수집

### 특히 우수한 문항

| 문항 | 이유 | 예비조사 연결 |
|------|------|--------------|
| **C7** | 학습맥락 제공 습관 자기 평가 | 72.3% 문제의 학생 관점 확인 ⭐⭐⭐ |
| **C8-C9** | 메타인지 (QAC 불가능) | 자기주도 학습 측정 ⭐⭐⭐ |
| **Part 1** | AI 역질문 경험 + 태도 변화 | Agent vs Freepass 명확한 대조 ⭐⭐⭐ |
| **Part 2** | 학습 방식 변화 자각 | 명료화 효과의 내재화 확인 ⭐⭐⭐ |
| **Part 4** | 구체적 에피소드 | "아하 모멘트" 발견 ⭐⭐⭐ |

### 권고사항: 현재 설문지 유지 ✅

**변경 불필요 이유:**
1. QAC와 중복 최소화 (75%는 완전 보완, 25%는 의미 있는 중복)
2. RQ2(학습 경험 향상) 검증에 최적화
3. 문항 수 적절 (20개, 15-20분)
4. 정성 데이터 충분 (Part 1-5의 가이드 질문)

**추가 분석 제안:**
- QAC A3(학습맥락) 점수와 설문 C7(맥락 제공 습관) 점수 간 상관 분석
- QAC C2(학습 지원) 점수와 설문 B1-B3 점수 간 상관 분석
- 높은 QAC 점수를 받은 학생의 정성 응답 패턴 질적 분석

## 7. 논문 작성 시 QAC-설문 통합 서술 예시

### 연구 방법 섹션

```markdown
본 연구는 MAICE 시스템의 효과를 다각도로 검증하기 위해 
**QAC 체크리스트(객관적 대화 품질)**와 **학생 사후 설문지(주관적 경험)**를 
상호 보완적으로 활용하였다.

QAC 체크리스트는 세션 로그를 기반으로 질문-답변-맥락의 객관적 품질을 
측정하여 RQ1(질문 품질 개선)을 검증하였으며, 학생 설문지는 QAC로 
측정할 수 없는 학습 경험, 메타인지적 자기 인식, 학습 성과를 측정하여 
RQ2(학습 경험 향상)를 검증하였다.

특히 설문 문항 C7("질문할 때 내 상황이나 배경 정보를 함께 제시한다")은 
예비 조사에서 발견된 학습맥락 부재 72.3% 문제를 학생의 자기 인식 
관점에서 확인하며, QAC A3(학습 맥락 적용) 점수와 삼각 검증을 
가능하게 한다.
```

### 결과 섹션

```markdown
객관적 대화 품질(QAC)과 주관적 학습 경험(설문) 모두에서 Agent 그룹이 
Freepass 그룹보다 우수한 것으로 나타났다:

- QAC A3(학습맥락): Agent 4.2 vs Freepass 3.1 (p < .001, d = 0.8)
- 설문 C7(맥락 제공 습관): Agent 4.0 vs Freepass 3.3 (p < .01, d = 0.6)
- 상관 분석: QAC A3와 설문 C7 간 r = 0.54 (p < .001)

이는 명료화 프로세스가 객관적으로 질문 품질을 개선했을 뿐만 아니라,
```
