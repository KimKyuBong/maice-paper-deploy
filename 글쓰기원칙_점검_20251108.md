## 🔍 **초록 점검 (22줄)**

### **현재 문장 (지나치게 긴 문장)**

```
고등학교 2학년 58명을 대상으로 수학적 귀납법 단원에서 무작위 배정 A/B 테스트를 실시한 결과(2024.10.21~11.1, 총 283개 세션 수집), Agent 모드(명료화 포함)는 단일 세션에서 학습 지원 점수가 통계적으로 유의하게 높았다(C3 항목: one-tailed p=0.034, Cohen's d=0.282). 전체 283개 세션의 회귀 분석 결과(3개 AI 모델 평균 점수 기준), Agent 모드는 세션 반복 시 +0.29점/세션의 증가 추세를 보인 반면(p=0.096, 한계적 유의), Freepass 모드는 +0.05점/세션의 약한 증가를 보였다(p=0.641). 두 기울기의 차이는 통계적으로 유의하지 않았다(p=0.253). 특히 학업 하위권(객관식 하위 33%)에서 AI 모델 평가에서는 Agent가 +0.54점/세션, Freepass가 +0.09점/세션의 증가 추세를 보였으나 통계적으로 유의하지 않았으며(p=0.245), 교사 평가(4명 평균, n=32)에서는 Agent가 세션당 +2.79점, Freepass가 -2.22점의 변화를 보여 세션 반복에 따른 기울기 차이가 통계적으로 유의하였다(+5.02점/세션, p=0.011). 실제 관찰된 평균 점수 차이는 약 5점이었다.
```

**문제점**:
1. 🔴 **한 문단에 6개 문장** - 너무 복잡
2. 🔴 **수동태 과다**: "실시한 결과", "관찰된"
3. 🔴 **명사화**: "증가 추세", "변화"
4. 🔴 **간결성 부족**: 통계 정보가 과도하게 집중

---

### **✅ 수정안 (간결하고 능동적)**

```
**연구 설계**
고등학교 2학년 58명을 대상으로 수학적 귀납법 단원에서 무작위 배정 A/B 테스트를 실시하였다(2024.10.21~11.1, 총 283개 세션).

**단일 세션 효과**
Agent 모드(명료화 포함)는 단일 세션에서 학습 지원 점수가 Freepass 모드보다 통계적으로 유의하게 높았다(C3: p=0.034, d=0.282).

**누적 효과 (AI 평가)**
전체 283개 세션의 회귀 분석 결과(3개 AI 모델 평균), Agent는 세션당 +0.29점씩 증가한 반면(p=0.096), Freepass는 +0.05점 증가하였다(p=0.641). 두 기울기 차이는 통계적으로 유의하지 않았다(p=0.253).

**학업 하위권 효과**
학업 하위권(객관식 하위 33%)에서 교사 평가(n=32)는 Agent와 Freepass 간 통계적으로 유의한 차이를 확인하였다(+5.02점/세션, p=0.011). 실제 평균 점수 차이는 약 5점이었다.
```

### **4. 표와 그림의 활용** ⚠️

**현재 상태**:
- 초록에 통계 수치가 과도하게 나열됨
- 시각 자료 없이 텍스트만으로 복잡한 결과 전달

**개선안**:
```
초록에는 핵심 결과만 간단히 제시하고,
상세한 통계는 본문의 표로 제시
```

**예시 표**:
```markdown
| 분석 | Agent | Freepass | 차이 | p값 |

| 단일 효과 | 3.74 | 3.34 | +0.40 | 0.034* |
| 누적(AI) | +0.29/회 | +0.05/회 | +0.24 | 0.253 |
| 하위(교사) | +2.79/회 | -2.22/회 | +5.02 | 0.011* |
```
