    📊 핵심 통계 결과 요약¶

    RQ1: 질문 품질 개선¶

    • 질문 점수 증가: Agent +2.18 vs Freepass +0.52
    • p=.043, Cohen's d=0.307 (중간 효과) ✅

    RQ2: 학습 경험 향상¶

    • C3 학습 지원: Agent 3.74 vs Freepass 3.34
    • one-tailed p=.034, Cohen's d=0.275 (작은-중간 효과) ✅

    RQ3: 학습 효과 개선¶

    • 세션 증가폭: Agent +0.63 vs Freepass -0.36
    • p=.364, Cohen's d=0.298 (경계선)

    핵심 발견: 하위권 학생 효과 ⭐⭐⭐¶

    • 객관식 하위 33%: Agent +0.91 vs Freepass -0.70
    • p=.040, Cohen's d=1.204 (매우 큰 효과) ✅
