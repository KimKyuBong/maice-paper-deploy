### 1. 중복 제거
- **표에 통계 → 본문에는 통계 반복 X**

- 학술 논문 체계 강화

- 표: 정량 데이터 요약

## 📝 수정 원칙 (향후 적용)

### 표와 본문의 올바른 관계

**표 (Table)**:
- 정량 데이터 요약
- 통계, 비율, 점수, 개수
- 한눈에 비교 가능한 구조

**본문 (Text)**:
- 표를 참조하는 간단한 요약문
- 정성적 설명 (사례, 해석, 함의)
- 표에 없는 맥락 정보

### 나쁜 예 (중복)

```
[표] 맥락 없는 질문: 72.3%

본문: "전체 질문 중 72.3%가 맥락 없는 질문이었다..."
```

### 좋은 예 (역할 분담)

```
[표] 맥락 없는 질문: 72.3%

본문: "[표 X]에서 확인할 수 있듯이, 학습 맥락 부재가 가장 심각했다.
```
