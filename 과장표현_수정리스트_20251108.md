### **2. "5배 이상 빠른 학습 증가율"**
**위치**: 24줄
```
Agent가 Freepass보다 5배 이상 빠른 학습 증가율을 보임을 실증하였다
```
**문제**: 
- 통계적으로 유의하지 않음 (p=0.253)
- "실증"은 과장

**수정안**:
```
Agent가 Freepass보다 더 빠른 학습 증가 추세를 보였다(통계적으로 유의하지 않음)
```

### **3. "프리패스 방식의 근본적 한계"**
**위치**: 45줄
```
프리패스 방식의 근본적 한계(반복 사용 시 점수 감소) 규명
```
**문제**: 
- "근본적 한계"는 과도한 표현

**수정안**:
```
프리패스 방식의 제한점 확인 (반복 사용 시 증가 추세 약함)
```

### **6. "통계적으로 유의한 효과(p=0.040, d=1.204)"**
**위치**: 50줄
```
학업 하위권 학생에게 통계적으로 유의한 효과(p=0.040, d=1.204)
```
**문제**: 
- 이는 40명 분석의 잘못된 수치
- 실제 전체 분석에서는 p=0.245 (유의하지 않음)
- 교사 평가에서만 p=0.011 (유의)

**수정안**:
```
학업 하위권 학생에게 실용적 효과 (교사 평가 p=0.011)
```

## ✅ **수정 원칙**

### **1. 통계적 정확성**
- p>0.05 → "유의하지 않음" 명시
- "실증", "검증" → "확인", "관찰" 
- 효과 크기 과장 금지

### **2. 주장의 범위**
- "패러다임" → "설계 전환"
- "근본적" → "제한점"
- "새로운" → "대안적"

### **3. 감정적 표현 제거**
- "급속히" → "증가"
- "압도적" → "더 큰"
- "획기적" → 삭제

### **4. 한정어 추가**
- "~을 시사한다"
- "~의 가능성을 제시한다"
- "~을 관찰하였다"
