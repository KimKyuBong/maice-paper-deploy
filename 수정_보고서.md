## 1. RQ1 수정 - 연구 문제와 실제 결과의 불일치 해소

### 수정 내용

**수정 전**:
- RQ1: "질문 품질 개선 검증 (핵심 메커니즘)"
- 실제 결과: A1(수학적 전문성), A2(질문 구조화)는 유의미한 차이 없음, A3(학습 맥락)은 Freepass가 우수

**수정 후**:
- RQ1: "학습 과정 지원 효과 검증 (핵심 메커니즘)"
- 실제 결과에 맞게 C2(학습 과정 지원)를 핵심 검증 항목으로 명시
- 질문 품질 자체(A1, A2, A3)의 개선 가능성도 함께 탐색하도록 수정

### 수정된 부분

#### 1장 3절 연구 문제 (L66)
```markdown
수정 전:
**RQ1. 질문 품질 개선 검증 (핵심 메커니즘)**: MAICE의 질문 명료화 프로세스(Agent 모드)가 
학생의 질문 품질을 실질적으로 개선하는가? 특히 예비 조사에서 가장 심각한 문제로 확인된 
학습 맥락 적용, 질문 구조화, 수학적 전문성 영역에서 일반적인 즉시 답변 방식(Freepass 모드) 
대비 유의미한 개선이 나타나는가?

수정 후:
**RQ1. 학습 과정 지원 효과 검증 (핵심 메커니즘)**: MAICE의 질문 명료화 프로세스(Agent 모드)가 
학생의 학습 과정 지원 측면에서 실질적으로 개선하는가? 특히 사고 과정 유도, 이해도 확인, 
메타인지 촉진 등 학습 과정 지원(C2) 영역에서 일반적인 즉시 답변 방식(Freepass 모드) 
대비 유의미한 개선이 나타나는가? 또한 질문 품질 자체(A1, A2, A3)의 개선 가능성도 함께 탐색한다.
```

#### 1장 1절 예비 연구 (L26, L62)
- "질문 개선이 학습 효과 향상의 핵심 메커니즘" → "질문 품질이 답변 품질과 강한 상관관계를 가짐을 보여주었으며, 질문 개선의 중요성을 시사"
- 명료화 프로세스는 질문 품질 자체보다는 학습 과정 지원에서 차별적 효과를 보일 가능성을 탐색한다고 명시

#### 1장 2절 연구 목적 (L30, L42)
- "질문 개선이 학습 효과 향상의 핵심 메커니즘" → "질문 품질이 학습 효과와 밀접한 관련을 가짐을 시사"
- 학술적 연구 목적 2번을 "질문 품질 개선 효과 검증" → "학습 과정 지원 효과 검증"으로 수정

### 수정된 부분

#### 2.2절 마. 본 연구에의 적용 (L151-157)
```markdown
수정 전:
**2단계(문제 정의)의 구현인 명료화 프로세스**는 Dewey의 5단계를 실제 대화형 질문으로 변환한다:
- **1단계 (문제 인식)**: "이 개념을 공부하면서 어떤 부분이 가장 어렵거나 궁금하셨나요? 🤔"
- **2단계 (문제 정의)**: "지금까지 이해한 부분과 아직 헷갈리는 부분을 나누어볼까요?"
...
**실제 구현 특징**:
- K1-K4 질문 유형별로 맞춤형 명료화 질문 생성
- 매우 모호한 질문("수열 알려줘")에는 구체적인 선택지 제공
- 각 학생 응답마다 PASS/NEED_MORE 평가로 충분성 판단
- 최대 3회 시도 제한으로 과도한 명료화 방지

수정 후:
명료화 프로세스는 Dewey의 5단계를 실제 대화형 질문으로 변환하여, 학생이 스스로 질문을 
구조화하고 사고를 명료화하는 과정을 경험하도록 설계된다. 구체적인 구현 방식(명료화 질문
```
