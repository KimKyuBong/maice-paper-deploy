    V. 연구 결과¶

    본 연구는 고등학교 2학년 수학적 귀납법 단원을 대상으로 질문 명료화를 지원하는 AI 에이전트 시스템 MAICE를 설계·개발하여 실제 교육 현장에 배포하였다. 연구 대상은 2학년 수학 과목 수강생 58명(자퇴생 제외)이며, 이 중 55명(Agent 모드 27명, Freepass 모드 28명)이 MAICE를 실제로 이용하였다. 3주간 A/B 테스트를 통해 이들 55명 학생이 생성한 유효 세션 284개(Agent 115개, Freepass 169개)를 수집하였으며, 방법론적 한계를 상호 보완하기 위해 **LLM 평가(3개 모델)와 교사 평가를 병행**하여 명료화 효과를 검증하였다.

    ◯ 상호 보완: - LLM의 순환 논리 우려 → 교사가 검증 - 교사의 표본 부족 → LLM이 확장

    Dewey 5단계 진행: 11턴 대화를 통해 5단계 모두 시도됨. 명료화 3회가 문제 정의 → 가설 형성 → 추론 → 검증 과정을 촉진.

    사례 분석: 학생의 마지막 질문[23]에서 여전히 혼란이 관찰됨. Dewey 5단계가 모두 시도되었으나 최종 검증의 완전한 성공은 확인되지 않음.

    라. 질적 분석 종합¶

    (1) 교육적 특성¶

    질적 분석 대상 6개 세션(32-34점)의 공통 특징: - 지식 차원: 모두 K2(개념적) → K3(절차적) → K4(메타인지적) 수준 포함 - 대화 깊이: 평균 7.2턴 (5-11턴) - 명료화: 대화1, 대화6에서 명료화 질문 3회로 문제 정의 촉진

    (2) LLM 평가의 타당성¶

    LLM 평가는 교사 평가와 강한 상관(r=0.754, p<0.001)을 보여, 객관적 평가 지표로서의 타당성을 입증하였다. 질적 분석은 고득점 세션이 단순한 정답 제시를 넘어 **개념적·절차적·메타인지적 학습**을 거쳤음을 확인하였다.

    (3) 하위권 효과의 질적 메커니즘¶

    교사 평가에서 확인된 Q1 하위권 효과(Agent +6.32점, 표Ⅴ-12)의 메커니즘을 대화1(세션 156)을 통해 규명하였다:

    대화1 (Q1 하위권, 34.33점, 상위 0.4%): - 막연한 질문 → 명료화 3회 → 지식 차원 상승 (K2 → K3 → K4) - 5턴 대화를 통해 개념 이해, 절차 적용, 메타인지적 성찰 달성 - 메커니즘: 명료화 → 질문 구체화 → 맞춤 지원 → 지식 차원 고단계 도달

    4절 요약:

    질적 분석 대상 6개 세션의 특징: - 모두 K2 → K3 → K4 지식 차원 상승 경로 포함 - 대화1, 대화6에서 명료화가 문제 정의 촉진 - LLM 평가는 교사 평가와 강한 상관(r=0.754) - 하위권(Q1) 효과의 질적 메커니즘 규명 (대화1)

    V장 전체 요약:

    본 장에서는 MAICE 명료화 프로세스의 효과를 **4가지 증거를 통해 다각도로 검증**하였다.

    1. 객관적 평가 (LLM, N=284): - 전체 효과: C2(학습 지원) Agent 우수 (+0.28점, p=0.004, d=0.353) - 하위권(Q1) 효과: +2.26점, p=0.032, d=0.499 (중간 효과, 유의) - 3개 모델 평균, 높은 평가자 간 일치도 (ICC=0.656***)

    2. 전문가 평가 (교사, N=100): - 전체 효과: Agent 경향 (+2.25점, p=0.085, d=0.349, 경계적 유의) - 하위권(Q1) 효과: +6.32점, p=0.013, d=0.992 (매우 큰 효과) - LLM과 강한 상관 (r=0.754, p<0.001)

    3. 학습자 자기 평가 (설문, N=47): - AI 상호작용 품질: 4.37/5.0, 개념 이해: 4.39/5.0, 질문 능력: 4.22/5.0 - 68.4%가 질문 유도형(B 방식) 선호 (이유: "사고력 향상"(42%), "깊은 이해"(25%)) - 시스템 만족도: 4.62/5.0 (최고점)

    4. 질적 분석 (6개 우수 세션): - 지식 차원: 모두 K2(개념적) → K3(절차적) → K4(메타인지적) 수준 포함 - Dewey 반성적 사고: 대화6은 5단계 모두 시도 (단, 최종 검증에서 학생 혼란 지속) - 하위권 메커니즘: 대화1(Q1 하위권)에서 명료화 → 지식 차원 상승 (K2 → K3 → K4) → 고득점 - LLM 평가 타당성: 교사 평가와 강한 상관(r=0.754***)으로 객관성 입증

    수렴 패턴: - 양적 증거(LLM·교사 점수)와 질적 증거(지식 차원 분석)가 일관 - 객관적 평가와 주관적 체감(학생 설문)이 수렴 - 특히 **하위권 효과**가 모든 증거에서 일관 (LLM +2.26점, 교사 +6.32점, 대화1에서 K2→K3→K4)
