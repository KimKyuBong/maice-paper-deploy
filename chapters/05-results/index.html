    V. 연구 결과¶

    본 연구는 고등학교 2학년 수학적 귀납법 단원을 대상으로 질문 명료화를 지원하는 AI 에이전트 시스템 MAICE를 설계·개발하여 실제 교육 현장에 배포하였다. 3주간 A/B 테스트를 통해 284개 유효 세션을 수집하였으며, 방법론적 한계를 상호 보완하기 위해 **LLM 평가(3개 모델)와 교사 평가를 병행**하여 명료화 효과를 검증하였다.

    1. 연구 실행 및 데이터 수집¶

    가. 시스템 배포¶

    MAICE 시스템을 실제 고등학교에 배포하였다 (3주, N=58, 시스템 가동률 99.2%).

    나. 데이터 수집 현황¶

    [표Ⅴ-1] 수집 데이터 현황

    구분 Agent Freepass 전체
    세션 수 115 169 284
    학생 수 28 30 58
    1인당 평균 4.1 5.6 4.9

    주: LLM 3개 모델 평가 공통 세션 기준

    다. 사전 동질성 검증¶

    두 집단 간 중간고사 성적 차이 없음 (총점: t=1.18, p=.242; V-2절 표Ⅴ-2).

    주의사항: - Agent 모드 평균 57.71점, Freepass 모드 평균 63.47점으로 5.93점 차이가 있었으나 통계적으로 유의하지 않음(p=0.242) - 이 차이가 결과에 미친 영향을 완전히 배제할 수 없으므로, 해석 시 주의가 필요함 - 향후 연구에서는 ANCOVA(공변량 분석)를 통해 사전점수를 통제한 효과 분석을 권장함

    라. 명료화 프로세스 작동 확인¶

    Agent 모드 118개 세션 중 98개(83.1%)에서 명료화 질문이 수행되었다.

    명료화 횟수 3회

    [표Ⅴ-24] 대화6 (세션 75) Dewey 5단계 추적

    단계 실제 대화 턴 비고
    1. 문제 인식 "(k+1)! 을 다르게 나타낼 방법이 있어?" 0 문제 제기
    2. 문제 정의 명료화: "곱셈식으로 전개? k!으로 표현?" 4 명료화 질문
    3. 가설 형성 학생: "k!로 표현하는 방법" 탐색 5-18 여러 문제 제시 및 탐색
    4. 추론 전개 "(k+1)*k! > (k+1)*2^k 관계 확인" 17-22 부등식 관계 확인
    5. 검증 "(k+1)*k! > (k+1)*2^k가 어떻게 (k+1)*2k>2(k+1)이 되는거야" 23 학생 혼란 지속

    교육적 의의: - 11턴의 긴 대화를 통해 Dewey 5단계 모두 진행 - 첫 질문 "(k+1)! 표현"에서 마지막 "부등식 전개 관계 검증 시도"까지 질문 진화 명확히 관찰 - 명료화 3회가 문제 정의 → 가설 형성 → 추론 → 검증 과정을 체계적으로 촉진 - 막연한 표현 질문에서 구체적 부등식 관계 확인으로 사고 심화

    한계: - 학생의 마지막 질문[23] "(k+1)*k! > (k+1)*2^k가 어떻게 (k+1)*2k>2(k+1)이 되는거야"는 여전히 혼란을 보여주므로, 검증 단계가 완전히 완료되었다고 보기는 어렵다 - 이는 Dewey 5단계가 모두 진행되었으나, 최종 검증이 완전히 성공한 것은 아님을 시사
